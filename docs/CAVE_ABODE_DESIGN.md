# 洞府系统设计草案

## 一、解锁条件

### 1.1 拆分解锁

洞府系统分两阶段解锁，避免前期流失：

| 阶段 | 解锁条件 | 功能 |
|------|----------|------|
| **简易修行室** | 炼气 5 层 / 筑基期，坊市租赁 | 修炼速率 ×1.1 + 基础仓库 30 格 |
| **正式洞府** | 金丹期（realm 2） | 全部功能：修炼室/灵田/炼丹/炼器/阵法/悟道 |

**简易修行室**：
- 在坊市中花费灵石租用（日租 50 灵石 或 月租 1000 灵石）
- 提供基础的修炼加成和储物空间
- 不可升级、不可建造灵田/炼丹室等高级模块
- 是玩家在金丹之前的"过渡容身之所"

**正式洞府**：

| 条件 | 说明 |
|------|------|
| 境界 | **金丹期**（realm 2）解锁洞府功能 |
| 触发 | 金丹期后首次进入任意地图房间时，弹出引导："你已凝结金丹，可与天地共鸣，是否在此开辟洞府？" |
| 数量 | 每人最多 **1 座主洞府** + 后期可建 **3 座分府**（大乘期解锁分府） |

**为什么分两阶段？**
- 炼气 5 层 ∼ 金丹期约 5-10 小时在线时间。在此之前无"家"的阶段太漫长
- 简易修行室给玩家归属感，同时激励尽早冲击金丹

## 二、选址：在地图上建造

### 2.1 可建位置

地图上的**空房间**（无 NPC、无资源、无事件）可被占领为洞府。不同类型房间灵气浓度不同：

| 房间类型 | 灵气浓度 | 特征 | 灵脉上限 |
|----------|----------|------|----------|
| 荒山野岭 | 1.0× | 最常见，地图外缘 | 良品 |
| 灵气节点 | 1.5× | 地图上有淡金色标记 | 灵品 |
| 灵脉交汇 | 2.0× | 稀有，紫气标记 | 宝品 |
| 上古遗迹 | 3.0× | 极稀有，需探索发现 | 仙品 |

选址后不可更改（但可后期迁移，损失部分建筑等级）。

### 2.2 建造流程

```
1. 金丹期后，在地图空房间点击「开辟洞府」
  ——或从底部菜单「洞府」点击（金丹期前显示"金丹期后开启"）
2. 弹出确认 → 消耗 5000 灵石 + 10 块青石
3. 洞府建立 → 该房间在地图上标记为「洞府」
4. 底部「洞府」菜单项激活，可随时进入洞府面板
5. 从任意位置可「传送回府」（每日免费 1 次，后续消耗灵石）
```

**双入口设计**：底部菜单「洞府」+ 地图上洞府房间点击，两个入口进入同一洞府面板。

## 三、洞府功能模块

### 3.1 总览

| 模块 | 功能 | 解锁 |
|------|------|------|
| 修炼室 | 离线修炼安全区，速率 ×1.2 | 默认 |
| 仓库 | 独立大容量储物（初始 100 格） | 默认 |
| 灵田 | 种植灵草，自动生长，按时收获 | 金丹期 + 500 灵石 |
| 炼丹室 | 炼丹炉，提升炼丹成功率 +10% | 元婴期 + 2000 灵石 |
| 炼器室 | 炼器台，提升炼器评分 +5% | 元婴期 + 2000 灵石 |
| 阵法 | 防御外敌/隐藏气息 | 化神期 + 5000 灵石 |
| 悟道室 | 概率触发顿悟，获永久增益 | 合体期 + 1 万灵石 |
| 灵脉 | 升级地下灵脉，提升整体灵气浓度 | 各阶段 |
| 道侣居 | 道侣 NPC 入驻，提供修炼加成 | 合体期 + 道侣关系 |

### 3.2 修炼室

```
离线修炼保护：
  - 在洞府修炼室离线，不会被其他玩家/NPC 攻击
  - 修炼速率 ×1.2（洞府灵气汇聚）
  - 可消耗灵石升级：Lv1(1.2×) → Lv2(1.4×) → Lv3(1.6×) → Lv4(2.0×)

升级消耗：
  Lv1→Lv2: 1 万灵石 + 灵品材料 ×10
  Lv2→Lv3: 5 万灵石 + 宝品材料 ×10
  Lv3→Lv4: 20 万灵石 + 珍品材料 ×10
```

### 3.3 仓库

```
独立于随身背包的洞府仓库：
  - 初始 100 格（远大于初始 50 格背包）
  - 可升级扩容：+50/+100/+200/+400 格
  - 炼丹/炼器直接使用仓库材料（不必带在身上）
  - 安全：即使被击杀，仓库物品不掉落
```

### 3.4 灵田

```
种植系统：
  - 初始 4 块灵田，可升级到 16 块
  - 种植灵草种子 → 现实时间成熟（品阶越高越久）
  - 成熟后采集获得材料（数量 = 基准 × 灵脉加成）

成熟时间参考：
  凡品: 5 分钟    良品: 15 分钟
  灵品: 1 小时    宝品: 4 小时
  珍品: 12 小时   仙品: 48 小时

灵田升级：
  Lv1(4 块): 默认
  Lv2(8 块): 筑基期 + 2000 灵石
  Lv3(12 块): 元婴期 + 1 万灵石
  Lv4(16 块): 化神期 + 5 万灵石
```

### 3.5 阵法

```
防御阵法：
  - 化神期解锁
  - 消耗灵石维持运转（日均：品阶 × 100 灵石）
  - 防止 NPC 妖兽/邪修探查或袭击洞府
  - 被袭击时自动反击（伤害 = 阵法等级 × 500）

阵法等级：
  凡阵(1×): 防御  良阵(2×): 防御+幻象
  灵阵(3×): 防御+幻象+反击  宝阵(4×): 防御+幻象+反击+预警
  珍阵(5×): 完全隐藏洞府气息

隐藏气息：NPC 妖兽/邪修无法发现洞府位置，不会被随机袭击事件选中。
兽潮事件：每游戏年有一定概率触发区域兽潮，阵法等级越高，洞府受损越小。
```

### 3.6 灵脉升级

核心长期投入系统：

```
灵脉品质 → 所有模块效果 × 灵气浓度倍率

初始灵脉 = 选址时房间灵气浓度
升级后叠加：

| 灵脉等级 | 灵气浓度 | 升级消耗 | 境界要求 |
|----------|----------|----------|----------|
| 凡品 | 1.0× | 默认 | 金丹 |
| 良品 | 1.3× | 3 万灵石 | 金丹 |
| 灵品 | 1.6× | 10 万灵石 | 元婴 |
| 宝品 | 2.0× | 50 万灵石 | 化神 |
| 珍品 | 2.5× | 200 万灵石 | 合体 |
| 仙品 | 3.0× | 1000 万灵石 | 大乘 |
| 圣品 | 4.0× | 5000 万灵石 | 地仙 |
| 道品 | 5.5× | 2 亿灵石 | 天仙 |

灵脉升级后，修炼速率/灵田产出/炼丹品质全部受到倍率加成。
```

## 四、洞府事件

### 4.1 随机事件（NPC 驱动）

洞府不是静止的背景板——会有 NPC 登门：

| 事件 | 触发条件 | 结果 |
|------|----------|------|
| 散修求宿 | 灵脉 ≥ 良品 | 收留 → 获得随机材料回礼；拒绝 → 无事 |
| 邪修觊觎 | 灵脉 ≥ 灵品，阵法 < 灵阵 | 被偷走仓库随机材料 ×10 |
| 妖兽袭击 | 兽潮事件期间 | 阵法自动防御，受损需修理（消耗灵石） |
| 高人路过 | 灵脉 ≥ 宝品，极小概率 | 获得随机功法残页或炼丹配方 |
| 弟子慕名 | 灵脉 ≥ 珍品，道心 ≥ 60 | 可收为弟子，入驻洞府提供修炼加成 |
| 旧友登门 | 曾帮助过的 NPC | 带来交易机会或秘境线索 |

### 4.2 天道赐福（离线激励）

- 每离线 24 小时（现实时间），回到洞府时触发一次「洞府灵气」结算
- 修炼室中的离线修炼额外 +10%（叠加到离线 0.6× 之上 → 0.66×）
- 灵田作物成熟自动收获入库（不会烂在地里）

### 4.3 洞府日志

每次回到洞府时，面板顶部显示上次离开至今的**洞府日志**：

```
「洞府日志 · 天逆历 3 年 7 月 15 日 酉时」

· 灵田「天灵草」已成熟，自动收获 ×24
· 一名散修前来求宿，留宿一晚后离去，留下 3 块玄铁矿作为谢礼
· 洞府灵气氤氲，修炼室为你积累了 1,280 点修为
· 阵法感知到东方有妖兽气息，暂未靠近
```

日志最多保留最近 10 条，旧的自动清除。每条事件用 1-2 行文字简述，不弹窗，纯被动阅读——给离线回来的玩家一种"这个世界在我离开时仍然运转"的感觉。

## 五、悟道室

### 5.1 功能

修炼室之外，洞府可建**悟道室**（合体期解锁）：

- 不是加速修炼，而是**触发顿悟**
- 在悟道室中，每现实小时有极小概率（道心/10000）触发顿悟事件
- 顿悟：随机获得一项永久效果，如：
  - 「剑意 +1」— 剑系技能伤害永久 +2%
  - 「丹道精进」— 炼丹毒性降低 5%
  - 「阵法心得」— 下一次布阵消耗减半
  - 「道心微澜」— 道心 +1（道心 ≤ 95 时）

悟道室等级越高，顿悟概率越高（×1.0 / ×1.5 / ×2.0 / ×3.0）。但悟道室不能堆到"挂机必出顿悟"——上限约每天 1-2 次小顿悟。

### 5.2 为什么要有悟道室

修炼数值（灵气积累）和突破概率是"确定性的成长"，而悟道室提供"不确定性的惊喜"——两种节奏互补。玩家每次回府都可能发现"咦，我悟了点什么"，这种微小正反馈是文字游戏的核心乐趣。

## 六、洞府命名与描述

### 6.1 命名

- 建造洞府时，玩家可自定名称（如「青云居」「剑庐」「丹霞洞」）
- 名称显示在地图房间标签上 + 洞府面板顶部
- 可随时更名（免费）

### 6.2 文字描述随等级演化

作为文字游戏，洞府的"外观"靠描述文本体现。每次升级模块/灵脉时，房间描述自动更新：

```
灵脉·凡品: "一处简陋的石洞，灵气稀薄，勉强可居。"
灵脉·良品: "洞壁透出淡淡荧光，灵气渐聚，修炼事半功倍。"
灵脉·灵品: "洞府内雾气氤氲，灵石镶嵌壁间，灵气浓郁如液。"
灵脉·宝品: "洞天福地初具雏形，紫气东来，宛若仙境一隅。"
灵脉·珍品: "祥云缭绕，灵鹤来栖，洞府自成一方小世界。"
灵脉·仙品: "仙光普照，大道法则显化，凡入此府者皆感天地共鸣。"
灵脉·圣品: "虚空生莲，道韵流转，洞府即道场。"
灵脉·道品: "洞府已非洞府——是道本身在此地的一缕投影。"
```

同样，修炼室/炼丹室/灵田各有对应描述文本，随等级变化。

## 七、洞府迁移

| 操作 | 代价 | 保留 |
|------|------|------|
| 废弃洞府 | 建筑等级重置为 0，灵脉等级 -2，材料不返还 | 仓库物品全保留 |
| 迁移洞府 | 消耗灵石 = 灵脉等级 × 10 万，在新地点重建，灵脉等级 -1，建筑等级保留 | 仓库物品全保留 |

## 八、洞府与其他系统的关系

| 系统 | 洞府影响 |
|------|----------|
| 修炼 | 修炼室加成 × 灵脉倍率 × 选址灵气浓度 |
| 炼丹 | 炼丹室加成 × 灵脉倍率（灵脉越高，丹药品质越高） |
| 炼器 | 炼器室加成 × 灵脉倍率 |
| 背包 | 仓库与随身背包分离，炼器/炼丹直接消耗仓库材料 |
| 地图 | 洞府房间在地图上特殊标记，"传送回府"可快速返回 |
| 时间 | 灵田按现实时间生长，离线也计算 |
| 战斗 | 洞府有阵法保护，被 NPC 袭击时自动反击。在洞府内修炼不可被 NPC 攻击 |
| NPC | 道侣/弟子 NPC 可入驻，提供持续 buff；散修/邪修/高人随机登门事件 |
| 宗门 | 宗门洞府与个人洞府独立（后期扩展） |
