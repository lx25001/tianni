## 手机端设计总纲

### 平台定位

这是一款**手机端文字修仙游戏**。不是 PC 动作游戏，不是 3D 开放世界。所有的操作通过**图形 UI + 按钮**完成——点按、滑动选择、长按确认。没有键盘搓招、没有鼠标画符、没有精确到毫秒的时机判断。

### 交互原则

| 原则 | 说明 |
|------|------|
| 单手可玩 | 核心操作区域在屏幕下半部分，拇指可及 |
| 按钮驱动 | 所有行动通过按钮触发，不依赖手势精度 |
| 文字即画面 | 战斗、炼丹、探索的核心反馈是**文字描述**——精美的 UI 框内，一段文字比一个 3D 动画更有沉浸感 |
| 碎片化时间 | 单次操作 15 秒内完成。一场战斗 1-3 分钟。一次炼丹 30 秒。随时可暂停、可退出 |
| 离线不亏 | 修炼自动进行。离线时积累修炼进度，上线时一次性结算——"你闭关了一夜（8 小时），游戏里已经过去了 4 个月" |

### 屏幕布局

```
┌─────────────────────────────┐
│         状态栏               │  ← 境界 / 气 / 灵石 / 神识
│─────────────────────────────│
│                             │
│         主内容区             │  ← 文字描述、事件、战斗画面
│      (占屏幕 60%)           │     功法插图、NPC 立绘
│                             │
│─────────────────────────────│
│         操作区               │  ← 3-5 个核心按钮
│      (占屏幕 30%)           │     底部导航栏
│─────────────────────────────│
│  探索 │ 修炼 │ 坊市 │ 洞府 │ 我  │  ← 5 个常驻标签
└─────────────────────────────┘
```

### 核心循环（手机端视角）

```
上线（3 分钟）→ 收离线修炼成果 → 清一波妖兽/做一笔交易 → 设下一个目标 → 下线
上线（15 分钟）→ 探索一个秘境 → 炼丹/炼器 → 和 NPC 聊几句 → 下线
上线（1 小时）→ 深入虚空、打 BOSS、参与坊市事件
```

没有"必须在线 X 小时"的内容。你决定投入多少时间，世界按你的节奏运转。

## 零、世界设定

### 0.1 世界观

**天逆大陆**——上古时期，天道崩碎，散落为无数碎片漂浮于混沌之中。每一块碎片自成一方小世界，灵气浓度、法则、物种各不相同。修仙者以"碎虚舟"穿梭于碎片之间。

没有人知道天道为何崩碎。有人说是远古大能的战争，有人说是天道自己选择了分裂——为了让每个生灵都能找到属于自己的道。

**平行宇宙法则**：天道崩碎不仅在空间上形成了无数碎片，在时间与因果上也分裂成了无数个**平行的宇宙**。每一个修士（玩家）的本地存档，就是一个独立的平行宇宙。在你的宇宙里，你劈开的山谷将永远存在，你覆灭的宗门将化为废墟，你杀死的 NPC 不会再复活。你在你的世界里拥有绝对的改变权——因为这是**唯属于你的因果线**。

当两个修士通过交易行交换物品时，那是两个平行宇宙之间唯一的因果交汇点。一件来自异世界的装备，承载着那个世界的记忆和历史——它在那里被谁打造、被谁使用、经历过什么。这些与你无关，但因为这件物品，你知道了：在另一个世界，事情不是这样的。

**你的故事**：你在一场"碎虚风暴"中醒来，身无长物，只有一本残破的《引气入体诀》和一块指向最近坊市的罗盘。你不认识任何人，没有人知道你是谁。你是这片大陆上一个全新的变量。

### 0.2 势力版图

| 势力 | 性质 | 立场 | 特色 |
|------|------|------|------|
| **太一道盟** | 正道联盟 | 秩序 | 最大势力，掌管大部分坊市，收税但不干涉内部事务 |
| **万妖岭** | 妖族领地 | 中立 | 妖兽材料的主要来源，人妖可交易但不可踏入深处 |
| **血煞宗** | 魔道第一宗 | 混乱 | 出产最强魔修功法，但修炼者被正道通缉 |
| **丹霞谷** | 炼丹师圣地 | 中立 | 垄断高阶丹方，想学炼丹必须获得谷主认可 |
| **铸剑城** | 炼器师行会 | 中立 | 天下名器的出产地，有最好的矿脉和地火 |
| **天机阁** | 情报组织 | 绝对中立 | 贩卖一切信息——NPC位置、功法出处、秘境入口，标价即所得 |
| **散修联盟** | 无门无派者 | 自由 | 没有任何要求，纯粹的交易互助组织，新人起步的第一站 |

### 0.3 境界

境界不是"目标"，是你在修炼过程中自然到达的状态。就像人会长大变老一样——引气入体久了，自然会筑基；筑基久了，自然结丹。没有人在终点等你，也没有证书要拿。

**二十四境界，每境九层**。总共 216 个小层次。每层的提升是渐进的——第 1 层和第 9 层的差距肉眼可见，但不会有什么"质的飞跃"非要你到了某层才能体验某个内容。层数只是你在当前境界中走了多远的标记，不是门禁。

```
凡人的你
    ↓ 第一次引气成功

—— 凡尘 ——
[01] 炼气 1-9 层    ← 学会用灵气，身体开始变化
[02] 筑基 1-9 层    ← 打好根基，决定你以后能走多远
[03] 金丹 1-9 层    ← 体内凝结金丹，灵气储量质的飞跃
[04] 元婴 1-9 层    ← 金丹裂开，里面有一个小小的你
[05] 化神 1-9 层    ← 神识可以离开身体，去肉眼看不清的地方
[06] 合体 1-9 层    ← 神识与肉身不再分彼此。"我有一个身体"→"我是这个身体"
[07] 大乘 1-9 层    ← 开始悟到天地法则。不是学来的，是自己知道的
[08] 渡劫 1-9 层    ← 天地对你的存在开始产生反应，每层都是一次小劫

[09] 地仙 1-9 层    ← 劫后余生。陆地逍遥，寿元大幅延长
[10] 天仙 1-9 层    ← 可遨游太虚，无需碎虚舟也能在碎片间穿行
[11] 真仙 1-9 层    ← 去伪存真。开始能看透事物的表象
[12] 玄仙 1-9 层    ← 法则的纹路清晰如掌中纹——你能"读"它们了
[13] 金仙 1-9 层    ← 金身不坏。肉身不再自然衰老，时间只在神识上留痕
[14] 太乙 1-9 层    ← 时空初握。可以创造属于自己的小片空间——领域
[15] 大罗 1-9 层    ← 万法归一。因果如蛛网般清晰可辨，时空不再是单向的
[16] 混元 1-9 层    ← 混沌即我。你开始理解天道的"源代码"——不是变强了，是角度变了

—— 道境 ——
[17] 鸿蒙 1-9 层    ← 站在道之初始，能看到天地初开时的回响

[19] 太初 1-9 层    ← 在"有"和"无"之间行走。存在和不存在的边界模糊了
```

每个境界突破时可能迎来**天劫**——可能是天雷，可能是心魔，可能是你年轻时杀过的人回来找你。天劫不是"考验"，只是天地对太强的存在产生的自然反应。就像你把一块巨石扔进湖里，水面必然会激起波浪。

没有人要求你突破。停留在筑基期两百年，种田养鱼，也是一种活法。

## 一、能力体系

### 1.1 没有职业，只有本事

你不是"剑修"、"丹师"或"符修"——你只是一个会一些本事的人。今天学了一本控火术，明天可能对画符感兴趣，后天突然想学种田。没有人给你贴标签，也没有"选错路线"这回事。

所有功法、技艺、能力都是独立的。你可以学任何东西，没有数量限制、没有点数分配、没有洗点。一个人同时擅长炼丹、剑术、阵法——就像一个真实的人可以有多种技能。

### 1.2 技能的网络而非技能树

功法之间不是树状依赖关系，而是**松散的前提网络**：

- 大部分功法只需要"能引气入体"这个最基本的前提（开局就会）
- 少部分功法有自然的前置——比如《三昧真火》需要先掌握基础的控火方法，因为你总不能跳过"怎么点火"直接学"怎么喷火"
- 但这些前置不是"职业路线"，只是常识性的学习顺序
- 一本功法不会锁死你学其他功法。学火的照样可以学水的，只是需要花两份时间

### 1.3 功法的六个维度

每本功法修炼后提供六个维度的加成。详细数值见 `SYSTEMS.md`。

| 维度 | 简称 | 影响 |
|------|------|------|
| **威** | POW | 伤害数值。直接提升攻击力和技能伤害倍率 |
| **妙** | FIN | 连招与暴击。提升暴击率和暴击倍率 |
| **固** | FOR | 防御与生存。提升 HP、防御、状态抗性 |
| **玄** | MYS | 特殊机制。解锁功法隐藏效果（时间减缓、分身、预判等） |
| **速** | SPD | 速度。提升气回复、闪避率、战斗中的行动顺序 |
| **悟** | COM | 修炼加成。提升本功法的熟练度增长速度和修炼速度 |

不存在六维全满的功法。每本功法总点数约 60，分配在不同维度上——有的功法威猛但笨重（威高、速低），有的功法灵动但脆弱（妙高、固低）。这正是"你擅长什么"的自然体现。

### 1.4 功法来源

| 来源 | 稀有度范围 | 特点 |
|------|-----------|------|
| 宗门藏经阁 | 凡品-宝品 | 拜入宗门后可用贡献点换，稳定但平庸 |
| NPC 传授 | 灵品-仙品 | 好感度满后触发传授任务 |
| 秘境/遗迹掉落 | 灵品-道品 | 随机，最刺激的来源 |
| 自创功法 | 取决于悟性 | 消耗"道韵"将多本功法融合，产出新功法 |
| 交易行购买 | 凡品-仙品 | 别人不要的，也许是你需要的 |
| 天机阁情报 | — | 告诉你某本功法的具体位置 |
| 杀人夺宝 | — | NPC 掉落其修炼的全部功法残卷 |

### 1.5 功法熟练度与蜕变

每本功法有熟练度条（0-100%），使用该功法的技能战斗会提升熟练度。达到 100% 后触发**"功法蜕变"事件**——可能的结果：

- 80%：威力提升 20%（通常）
- 10%：觉醒隐藏词条（如"对火系敌人额外 30% 伤害"）
- 5%：功法进阶为上位功法的一页残卷
- 5%：走火入魔，功法被封印 3 天

## 二、炼丹系统

### 2.1 丹方

丹方不是固定的配方表，而是**"条件组合"**：

```
《筑基丹》丹方：
  主药：任意灵品 3 阶以上草药 ×1
  辅药：任意灵品草药 ×2
  药引：妖兽内丹 ×1
  火候：文火 3 刻 → 武火 1 刻 → 文火收丹
  丹炉要求：灵品以上
  炼制技能要求：炼丹术 30
```

**自由度**：主药不同，产出筑基丹的附加属性不同——火焰草炼出的筑基丹带火抗，冰魄花炼出的带冰抗。玩家可以探索最佳组合。

### 2.2 丹药的六维品质

每颗丹药有六项指标（0-100）：

```
纯度 ← 影响效果倍率
毒性 ← 过高会产生"丹毒"debuff
稳定性 ← 低则可能炼出废丹或炸炉
灵气 ← 影响突破类丹药的成功率加成
药力 ← 持续时间 / 叠加上限
特效 ← 额外的随机效果（正面或负面）
```

**这是炼丹师的核心追求**：不是能不能炼出来，而是能不能炼出"完美丹"（纯度 95+ / 毒性 5- / 稳定性 90+）。

### 2.3 炼丹界面与操作

炼丹是**按钮驱动的选择过程**，不是实时温度控制。

```
┌─────────────────────────────┐
│  丹炉：灵品·紫金丹炉         │
│  丹方：筑基丹                │
│─────────────────────────────│
│                             │
│  你将火焰草投入丹炉，         │  ← 文字描述当前步骤
│  草药在炉中缓缓融化，         │
│  散发出微甜的焦香。           │
│  现在需要加入辅药...          │
│                             │
│  炉温：文火 ●○○  ← 可调     │
│  进度：主药已融 ██░░░        │
│─────────────────────────────│
│ [加辅药] [调大火] [调小火]    │  ← 操作按钮
│ [开炉]   [放弃]              │
└─────────────────────────────┘
```

**炼丹流程**（按钮操作）：

1. **选丹方** → 从已学会的丹方中选择
2. **选材料** → 从背包中挑选主药、辅药、药引（同一种丹方可以用不同材料，影响最终丹药的附加属性）
3. **点火** → 选择初始炉温（文火/中火/武火），不同草药适应不同温度
4. **炼制过程**（按钮控制）：
   - `[加料]` — 按丹方顺序投入材料。投料时机（炉温是否合适）影响纯度
   - `[调温]` — 三段温度切换（文火↔中火↔武火），每次切换需要少量灵石
   - `[开炉]` — 结束炼制，产出丹药。开早了 → 不熟；开晚了 → 焦了
   - `[放弃]` — 材料损毁，丹炉保留
5. **收丹** → 查看丹药的六维品质

**深度在哪**：每株草药的"耐热值"有 ±10% 浮动（不显示在 UI 上）。你投料时炉温是否接近它的耐热值 → 需要在炼制过程中通过文字提示判断。"草药边缘开始发黑"意味着温度太高。"药液平静如镜"意味着刚好。熟练的炼丹师能从文字中读出最佳时机。

## 三、炼器系统

### 3.1 装备槽位

```
武器 / 防具 / 饰品 ×2 / 法宝 ×2 / 飞剑(坐骑) / 储物戒 / 本命法宝
```

**本命法宝**是最特殊的槽位——只能绑定一次（更换需要"碎丹重修"级别的代价），会随玩家境界成长而自动成长。

### 3.2 炼器流程与界面

炼器是**分段选择 + 按钮确认**的过程，每步一个界面：

```
┌─────────────────────────────┐
│  炼器：长剑（未命名）         │
│─────────────────────────────│
│  当前步骤：淬火              │  ← 进度指示
│  ████░░░░ 步骤 4/7          │
│─────────────────────────────│
│                             │
│  你将烧红的剑身浸入寒潭。     │  ← 文字描述
│  剑身与水接触的瞬间，         │
│  白雾弥漫，发出刺耳的嘶嘶声。  │
│  淬火效果：A 级               │  ← 本步评分
│                             │
│─────────────────────────────│
│ [下一步：附灵]  [重做此步]    │  ← 每步可重做（消耗材料）
└─────────────────────────────┘
```

七步流程（每步一个界面/一组按钮）：

1. **淬炼** → 选淬炼方法（火炼/水炼/雷炼），不同材料适合不同方法
2. **塑形** → 选装备类型（剑/刀/甲/饰...）和大致形态，决定基础属性分布
3. **淬火** → 选淬火介质（寒泉/灵液/兽血），影响耐久和品级上限
4. **附灵** → 选灵魂碎片（或不选），赋予特效
5. **开锋** → 决定伤害/防御的最终数值，可以选择"激进开锋"（高伤害低耐久）或"保守开锋"（反之）
6. **命名** → 自由输入名字，留空则自动生成

每步评分（D~S）+ 材料品级 + 炼器技能 → 最终装备品级。不满意某步可以`[重做]`（消耗材料但不消耗灵石）。

### 3.3 装备成长

装备不是固定的——使用过程中可以"蕴养"：

- 击杀一定数量敌人 → 武器获得"杀戮"词条
- 承受一定伤害 → 防具获得"不屈"词条
- 陪伴时间够长 → 装备产生"器灵"（AI 人格），偶尔和玩家对话

## 四、阵法和符箓

### 4.1 阵法系统

阵法是**按钮驱动的布局玩法**：

- 进入布阵界面 → 显示一个网格地图（你当前位置周围）
- 阵旗有数量限制（取决于阵法修为等级）
- 点击网格放置阵旗，每面阵旗占据一个格子
- 已放置的阵旗形成几何形状 → 自动识别并激活阵法效果：
  - 三角形 → 杀阵：踏入者在战斗开始时受到一次伤害
  - 正方形/矩形 → 困阵：踏入者前 3 回合无法逃跑
  - 直线排列（3 面以上）→ 剑气阵：踏入者每回合受到小额伤害
  - 圆形/环状 → 护阵：阵内友方每回合回复少量血量
- 布阵完成后，阵法效果在当前碎片地图上持续存在（直到被破坏或阵旗被收回）

**操作按钮**：
- `[放旗]` — 点击网格放置阵旗
- `[收旗]` — 点击已放置的阵旗收回
- `[激活]` — 确认布局，消耗灵石激活阵法
- `[预设]` — 保存当前布局为模板，下次一键放置

### 4.2 符箓系统

符箓是**一次性消耗品**，威力远超同境界技能：

- 符纸品级 × 朱砂品级 × 画符技能等级 = 符箓威力
- **画符界面**：主内容区显示符文的**轮廓线**（半透明），你需要用手指在屏幕上**沿着轮廓描一遍**
- 描得越贴近轮廓 → 符箓品质越高
- 偏离太多 → 符纸自燃，材料损失
- 不同符箓有不同的轮廓复杂度——最简单的"火球符"是一条 S 形曲线，最复杂的"天雷符"是三层嵌套的星形图案
- 画符技能等级越高 → 系统对偏离的容忍度越高（容错范围扩大）

**操作按钮**：
- `[画符]` — 开始描摹
- `[重画]` — 放弃当前描摹，换一张符纸（消耗材料）
- `[保存]` — 符箓存入背包
- `[批量]` — 连续画同一符箓，描摹一次后系统自动按上次精度复制（但有 ±5% 随机浮动）

**战术价值**：平时囤符，越级打 BOSS 时狂甩——一次性道具的爆发力是装备无法替代的。

## 五、探索与秘境

### 5.1 世界碎片机制

天逆大陆由无数"碎片"组成。每个碎片是一个**独立的地图种子**，大小不一（小如一个房间，大如一个城市）。

在手机端，每个碎片以**文字列表 + 可交互地点**呈现：

```
┌─────────────────────────────┐
│ 当前碎片：青云山              │
│ 灵气浓度：●●●○               │
│─────────────────────────────│
│ 可前往的地点：               │
│ [山巅灵泉] → 修炼效率 +20%   │  ← 点击进入该地点
│ [古剑遗迹] → 未探索          │
│ [废弃矿洞] → 矿脉（已开采）   │
│ [青云坊市] → 交易/任务       │
│ [无名洞府] → 属于玩家 XXX    │
│─────────────────────────────│
│ [探索周边] [碎虚舟航行]      │
└─────────────────────────────┘
```

发现新碎片的方式：

- 碎虚舟航行（需要灵石作为燃料，选择方向 → 航行 → 到达新碎片）
- 随机出现的"空间裂缝"（限时，免费但目的地随机）
- 其他玩家分享的坐标（输入坐标 → 航行）
- 天机阁购买坐标

### 5.2 秘境类型

秘境以**文字交互叙事**呈现——进入后，每到一个节点，主内容区显示场景文字描述，下方出现 2-4 个选择按钮。

| 秘境类型 | 进入方式 | 玩法 |
|----------|----------|------|
| **上古遗迹** | 碎片地图上点击进入 | 文字描述的 dungeon crawl——"前方有三条岔路，左侧传来水声，右侧有剑痕刻壁，中间漆黑一片。" 你选一条 |
| **传承秘境** | 需要特定功法/物品开启 | 试炼式——NPC 残像出题，你选择应对方式 |
| **杀阵** | 碎片地图上发现 | 阵中每步一个选择——"左转还是右转？"选错 → 触发陷阱（战斗或受伤） |
| **药园** | 碎片地图上随机刷新 | 采药界面——看到满园草药，选择先采哪株，守护妖兽随时可能回来 |
| **剑冢** | 需要剑法技能达到阈值 | 名剑择主——剑冢中的残剑有意志，你的选择会影响它是否认可你 |
| **心魔秘境** | 渡劫失败后出现 | 面对自己的选择——"那年你杀了那个散修。再来一次，你杀不杀？" |
| **限时裂隙** | 服务器推送（赛季内容） | 限定时间内的文字冒险，全服玩家各自探索，服务器统计事件结果 |

### 5.3 秘境中的抉择

没有最优解——每个选择按钮都对应不同的风险和收益：

```
┌─────────────────────────────┐
│  你站在一个岔路口。           │
│  左侧传来隐约的水声——         │
│  可能是地下灵泉，也可能       │
│  是某种水生妖兽的巢穴。       │
│                             │
│  右侧的石壁上布满了剑痕——     │
│  深浅不一，像是有人在此练剑   │
│  多年。剑痕中隐约有灵气残留。  │
│                             │
│  中间的路完全漆黑。           │
│─────────────────────────────│
│ [走向水声] [探查剑痕] [深入黑暗] │
│ [使用神识探查]（消耗神识 10）  │
└─────────────────────────────┘
```

`[使用神识探查]` 会给出一段额外文字描述作为线索——但消耗神识资源。用还是不用？

## 六、宗门系统

### 6.1 拜入宗门

每个宗门有：

- **入门条件**：境界、天赋、或者完成入门任务
- **门规**：每月贡献度要求、禁止事项（如禁止修炼魔功）
- **俸禄**：每月发放的灵石和丹药
- **藏经阁**：功法树的一部分在此解锁
- **宗门任务**：跑腿、采集、讨伐，赚贡献点

### 6.2 宗门内地位

```
外门弟子 → 内门弟子 → 真传弟子 → 长老 → 副掌门 → 掌门
```

地位提升靠**贡献度 + 实力 + NPC 关系**。不是纯数值门槛——需要和师傅/同门搞好关系（或干掉竞争者）。

### 6.3 玩家自建宗门

达到元婴期后，可以购买"山门令"建立自己的宗门：

- 选址（灵脉交汇处最优，但 NPC 宗门也会来抢）
- 建造建筑（藏经阁、丹房、演武场……），每栋建筑需要资源和时间
- 招募 NPC 弟子（散修 NPC 会来投奔，取决于宗门名声）
- 设定宗门规矩、税收、福利
- 宗门战：和其他 NPC 宗门争夺灵脉

**成为掌门的代价**：宗门日常事务会占用修炼时间。做甩手掌柜也可以，但宗门发展会很慢，而且可能被弟子篡位。

## 七、人际关系系统

### 7.1 NPC 的七情六欲

每个 NPC 有性格模型（五行人格 + 善恶值 + 贪嗔痴三毒数值），决定了他们如何对待玩家：

- **好感度**：从 -100（不共戴天）到 +100（生死之交）
- **信任度**：从 -100（你说的每个字我都怀疑）到 +100（你说是就是）
- **敬畏度**：你的实力远高于他时自动增长，影响 NPC 是否主动挑衅

### 7.2 可建立的NPC关系

| 关系 | 条件 | 好处 |
|------|------|------|
| 道友 | 好感度 30+ | 可组队，共享部分功法心得 |
| 师徒 | 拜师需要对方同意 | 传授独门功法，突破时护法 |
| 道侣 | 好感度 80+ 触发特殊事件 | 双修效率 +30%，共享部分属性加成 |
| 仇敌 | 好感度 -50- | 见面就动手，杀了爆装备 |
| 债主 | 借灵石不还 | NPC 派打手追债 |
| 恩人 | 救过 NPC 的命 | NPC 会在你危险时随机出现帮忙 |

### 7.3 NPC 记忆系统

NPC 会记住和你的互动历史：

```
"三年前在青云山，你抢了我一株千年灵芝。今天我结丹成功了，来找你算账。"
```

这创造了**"你做的事会产生长期后果"**的沉浸感。

## 八、异步 PvP 系统

### 8.1 挑战书

不搞实时对战（违背半联机原则），而是**挑战书机制**：

```
1. 玩家 A 向玩家 B 发起挑战（需要知道 B 的道号或 ID）
2. B 离线也可以——系统记录 B 的角色数据（镜像）
3. A 和 B 的 AI 镜像战斗，录像
4. B 上线后收到"你被 XX 挑战了，观看录像并回应？"
5. B 可以选择：接受结果 / 要求重赛 / 无视
```

### 8.2 坊市公告板

坊市入口有一块公告板，任何人都可以在上面贴条：

- 求购某种材料
- 组队探索某个秘境（留道号，上线后约时间）
- 悬赏某个 NPC 的人头
- 纯粹的无聊留言

公告板不做排名，不搞竞争。它只是这个世界里的一块木板，路过的人低头看一眼，然后继续走自己的路。

## 九、经济系统·扩展

### 9.1 物品品级

所有物品——装备、丹药、材料、功法——都有一个**品级**。品级不是"等级"，是物品在天地间的"分量"。

**八品制**，由低到高：

| 品级 | 颜色 | 含义 | 典型来源 |
|------|------|------|----------|
| 凡品 | 灰 | 寻常之物，毫无灵气 | 凡人世界、新手村 |
| 良品 | 白 | 略含灵气，可用但不值钱 | 炼气期日常产出 |
| 灵品 | 绿 | 真正的修仙物品，灵气充沛 | 筑基~金丹期主流 |
| 宝品 | 蓝 | 宝物级别，可作传家宝 | 金丹~元婴期追求 |
| 珍品 | 紫 | 珍稀之物，一件难求 | 元婴~化神期稀有掉落 |
| 仙品 | 金 | 仙家之物，凡间不可量产 | 化神以上秘境、完美炼制 |
| 圣品 | 橙 | 传说中的品级，每一件都有名字 | 特殊事件、禁术代价 |
| 道品 | 红 | 天地生成，非人力可为。每一件都是一个故事 | 天道碎片、终极奇遇 |

品级决定了物品的基础数值范围、交易价值、和识货 NPC 对它的反应。一个识货的 NPC 看到你腰间挂着一件圣品——他的眼睛会亮。不是贪婪，是敬意。

### 9.2 灵石的生命周期

```
灵石产出：
  ├── 修炼时自动吸收天地灵气凝聚（基础产出，人人都有）
  ├── 矿脉开采（需要找到矿脉 + 采矿技能）
  ├── 妖兽掉落
  └── NPC 任务奖励

灵石消耗：
  ├── 修炼加速（消耗灵石提高灵气吸收速度）
  ├── NPC 交易
  ├── 碎虚舟航行燃料
  ├── 交易行 5% 税费（系统回收，防通胀）
  ├── 宗门建设
  └── 渡劫时布阵消耗

核心设计：灵石有"系统回收"渠道（交易税），保证不会无限通胀。
```

### 9.2 坊市深度玩法

坊市不只是"打开一个列表"。它是一个**实际的场所**（离线渲染，联网时同步价格）：

- 摊位有限，先到先得
- 位置好的摊位（门口 vs 角落）曝光率不同
- NPC 会租摊位卖东西
- 玩家可以在坊市**摆地摊**：站着展示一件装备，路过的 NPC/玩家可以出价
- **拍卖行**：每周六晚上，稀有物品走拍卖而非一口价

### 9.3 职业经济人玩法

不修炼不打架，纯靠经济活动"成仙"是可行的：

```
策略举例：
  低买高卖：在 NPC 坊市买低价草药，炼成丹药后在玩家交易行高价卖出
  倒买倒卖：两个坊市的 NPC 定价不同，来回倒卖赚差价
  情报交易：探索到稀有资源点/秘境入口，把坐标卖给天机阁
  代工服务：接单帮人炼丹/炼器（通过交易行"定制单"功能）
  灵石借贷：借灵石给 NPC，收利息（有坏账风险）
```

## 十、更多玩法系统

### 10.1 妖兽/灵兽

**击杀妖兽**：
- 妖兽分 1-10 阶，对应修士的炼气到大乘
- 高阶妖兽有特殊攻击模式（需要背板）
- 妖兽材料是炼丹炼器的核心原料
- 兽潮事件：随机触发，大量妖兽冲击坊市/宗门，全服玩家共同抵御

**驯养灵兽**：
- 击败妖兽后，有小概率出现"可驯服"的幼崽
- 灵兽可以：协助战斗、帮忙采集、守家、当坐骑
- 灵兽有成长树，喂不同丹药进化成不同形态
- 灵兽之间可以**繁育**：两只高阶灵兽的后代可能继承父母的优秀词条

### 10.2 碎虚舟（坐骑/航行）

碎虚舟不只是坐骑，是**移动基地**：

- 基础功能：在碎片间穿梭
- 可升级：扩大船舱 → 移动仓库
- 可装饰：炼丹炉放船上 → 航行时也能炼丹
- 船战：在虚空中遭遇虚空兽或其他玩家的船（异步），船上的防御阵法自动战斗

**获取方式**：
- 基础版：主线 NPC 赠送（只能去最近的坊市）
- 进阶版：铸剑城购买 / 自己炼制
- 传说版：探索远古遗迹获得，自带特殊技能

### 10.3 奇遇系统

探索中随机触发"奇遇"：

| 奇遇类型 | 示例 |
|----------|------|
| 前辈洞府 | 发现一个陨落修士的洞府，继承遗产（也可能被夺舍） |
| 天降陨石 | 一颗陨石落到附近，全是稀有矿石 |
| 时空裂隙 | 短暂穿越到上古时期，亲眼见证某个历史事件 |
| 神秘商人 | 一个行踪不定的 NPC，卖的东西来自"其他世界" |
| 传承试炼 | 某个上古大能留下的考验，通过后获得其传承 |
| 因果事件 | 你之前做的事产生连锁反应——比如你杀的某个 NPC，他师傅来报仇了 |

奇遇的概率基于**探索频率**，不搞"挂机能触发"——鼓励玩家到处跑，而不是在一个地方挂机。

### 10.4 天气与天象

天气影响 gameplay：

| 天象 | 效果 |
|------|------|
| 灵气暴动 | 修炼速度 3x，但走火入魔概率 5x（赌狗狂欢） |
| 天狗食日 | 魔修功法的效果翻倍，正道功法减半 |
| 流星雨 | 野外大量刷新陨铁（炼器材料），引发争夺 |
| 血月 | 所有妖兽狂暴化，兽潮概率大幅增加 |
| 七彩祥云 | 渡劫成功率翻倍，全服玩家都在找地方突破 |
| 虚空风暴 | 碎虚舟无法航行，坊市交易暂停 |

天象是**全局服务器推送**的（所有人看到同一天象），创造"所有人都在做同一件事"的共同体验。

### 10.5 洞府系统

每个玩家可以拥有一处"洞府"（家园）：

- 选址：任意碎片中的任意位置（灵脉旁有加成）
- 功能建筑：修炼室、丹房、炼器房、灵兽园、药田
- 防御设施：护山大阵（玩家自己布的阵法）
- NPC 可以**登门拜访**：你的师傅/道友会上门聊天或求助
- 可被 NPC 攻击：如果你罪恶值太高，正道 NPC 会来攻打你的洞府
- 搬家：可以放弃洞府换地方，但建筑搬不走，只能拆了重建

### 10.6 传承与死亡

**死亡惩罚**：
- 普通死亡：当前境界经验归零，身上物品随机掉落 30%
- 有"替身符"：符箓消耗，免死一次
- 渡劫失败：掉一个小境界 + 积累"劫伤"

**传承机制**：
- 玩家可以主动选择"转世重修"：保留所有功法知识（功法熟练度不重置），但境界归零
- 转世后获得"宿慧"buff：修炼速度 +50%直到回到前世境界
- 可以转世 9 次，每次获得不同的"轮回印记"永久词条
- 可以转世多次，每次获得轮回印记（详见第二十五章）

## 十二、技术架构

### 12.1 基础定位

这是一款**以单机为主、可选的玩家间交易为辅**的游戏。90% 的内容离线可玩，联网仅用于玩家间交易。没有服务器托管游戏状态——你的角色、物品、进度全部存储在本地。

```
┌──────────────────────────────────────────────────┐
│                                                  │
│  玩家 ←→ NPC 交易（全程本地）                      │
│  ─────────────────────────                       │
│  • 不发生任何网络通信                               │
│  • 本地数据直接结算                                 │
│  • 价格由本地 NPC AI 模拟                           │
│  • 不需要任何校验——你改灵石改掉落都只影响你自己        │
│                                                  │
│  玩家 ←→ 玩家 交易（服务器轻量校验）                  │
│  ─────────────────────────                       │
│  • 服务器仅做两件事：防双花 + 防重复上架              │
│  • 不存储玩家存档、不验证物品来源、不检查境界合理性     │
│  • 交易记录只保留 TXID，不保留物品详情               │
│                                                  │
└──────────────────────────────────────────────────┘
```

### 12.2 服务器职责（极简）

```
┌─────────────────────────────────────┐
│          交易服务器                   │
│                                     │
│  仅 4 个 API：                       │
│  /api/trade/list      上架物品       │
│  /api/trade/search    搜索交易行      │
│  /api/trade/buy       购买物品       │
│  /api/trade/cancel    下架物品       │
│                                     │
│  赛季 API：                          │
│  /api/season/status   赛季状态       │
│                                     │
│  数据库（极简）：                     │
│  • 已使用的 TXID 列表（防双花）        │
│  • 当前上架物品 ID 列表（防重复上架）   │
│  • 赛季进度                          │
│                                     │
│  明确不存储：                         │
│  ✗ 玩家存档、灵石余额                  │
│  ✗ 物品属性、来源、指纹                │
│  ✗ NPC 数据、任何 gameplay 状态       │
└─────────────────────────────────────┘
```

### 12.3 玩家间交易流程

```
上架流程:
  1. 客户端生成物品唯一 ID（本地 UUID）→ 发送给服务器
  2. 服务器检查该 ID 是否已在架上/已售出 → 是则拒绝
  3. 服务器记录: (物品ID, 上架时间, 价格)
  4. 客户端本地移除物品 → 服务器标记为"已上架"

购买流程:
  1. 买家选择物品 → 服务器生成交易 TXID
  2. 买家发送签名的灵石消费声明 + TXID
  3. 服务器检查 TXID 是否已使用（防双花）→ 是则拒绝
  4. 扣除灵石 → 物品 ID 发给买家
  5. 买家本地生成物品（基于卖家上架时提供的属性 JSON）
  6. 服务器标记物品 ID 为"已售出"，永久记录 TXID
```

### 12.4 防作弊：只防影响他人的

**该防的**：

| 攻击 | 防御 |
|------|------|
| 同一物品重复上架 | 服务器维护已上架/已售出 ID 集合 |
| 灵石双花 | 每笔交易有唯一 TXID，已用即失效 |
| 截包重放 | nonce + 时间戳窗口（±30秒） |

**不该防的**（纯本地，不影响他人）：

| 行为 | 处理 |
|------|------|
| 修改灵石/NPC 互动 | 不管。你改再多也只能在本地用 |
| 修改掉落/属性 | 不管。能上架交易行就行——买家收到的是你上架时声明的属性。你吹牛说自己有把仙品剑，买家花灵石买到的是凡品——那是买家自己的判断问题 |
| 单机自嗨 | 完全不管 |

### 12.5 诸天万界集市（跨界交易）

交易行不仅是物品流通的场所，更是**平行宇宙间因果交汇的唯一节点**。每一件来自异世界的物品，都带着原世界的记忆而来。

**异世铭文**：
你买到的高品级装备，物品描述中会带有原宇宙的历史记录。不是干巴巴的"攻击 +50"，而是——"此剑由（玩家 XXX）所在世界的铸剑城城主打造，曾在兽潮中饮血三千，后在城主寿元耗尽后被其弟子变卖"。你手里握着的不是一件装备，是一段异世界的因果。

**执念偷渡**：
如果你在交易行买到了带有 NPC"寻回执念"的物品——原主在另一个宇宙无法跨界追杀。但这股执念不会凭空消失。它会**感染**你当前世界中的某个对应实体：

- 感染一只虚空妖兽 → 它开始追踪这件物品，在某次探索中突然袭击你
- 感染一个同类型 NPC → 他莫名觉得你身上这件东西"不应该是你的"，找你讨要或挑战
- 感染一个坊市鉴定师 → 他看到这件物品时，会说出原主的名字——"这是 XXX 的东西。他不在这里。但他知道东西在你手上。"

这不是惩罚——这是一段你花钱买来的**意外故事**。

**异界忆珠**：
你可以从交易行买到来自其他宇宙的 NPC 记忆。这会带来极具冲击力的认知错乱：

> 在你的世界里，青木宗宗主是一位德高望重、从不参与纷争的老者。
> 某天，你在交易行买了一颗忆珠——来自另一个宇宙。
> 你捏碎它。你看到那个宇宙里的青木宗宗主，在秘境中将队友推进深渊，只为独吞功法。
> 在你的世界里，他从未做过这件事。但他能做吗？
> 这个疑问不会改变他的属性。但你再也不会用同样的眼光看他了。

### 12.6 天道排斥（异常数据的叙事化处理）

既然是纯本地游戏，系统不干涉玩家使用修改器。但一旦异常物品进入交易行，服务器不封号、不禁售——而是**把反作弊做成一种叙事**。

**伪天道之物**：
如果交易服务器检测到上架物品的数值严重溢出（如词条超出自然上限、属性组合不可能合法生成），服务器给该物品打上不可见的标签——「天道排斥·污浊」。

**天谴反噬**（买家侧）：
买家购买并装备该物品后，在本地世界会被天道视为"入侵异数"：

- 下次渡劫时天雷威力翻 **5~10 倍**——天道在试图抹除不属于任何宇宙规则的异物
- 日常探索中高概率遭遇最高级别的虚空风暴、杀手截杀、或因果反噬事件
- 物品描述中出现一行红字："此物不应存于世间。持有者，好自为之。"

买家在购买时就看到了物品属性。他知道这东西"强得不正常"。他选择了购买——他选择了一个极限难度的挑战。天道排斥不是惩罚，是**玩家主动选择的 Hard Mode**。

如果买家能顶着天谴反噬存活——那他在这个世界里就是真正的逆天者。

## 十四、核心设计原则总结

1. **手机原生，文字驱动**：所有操作按钮化，反馈以文字描述为主——不是简化，是另一种表达深度
2. **本地为主，联网为辅**：90% 的内容离线可玩，联网只用于经济校验
3. **自由 = 可玩性**：没有"应该怎么玩"，只有"我想怎么玩"
4. **系统深，操作浅**：炼丹有六维品质但操作只需选料→点火→收丹几个按钮；战斗有功法克制但操作只需选择技能。深度在决策，不在手速
5. **选择有后果**：每个抉择都有长期影响，NPC 会记住你做过的事
6. **碎片时间友好**：单次操作 15 秒内完成，离线修炼自动积累，上线即结算
7. **不逼氪**：没有任何付费独占内容。外观可以有付费获取，但不影响数值平衡
8. **不作弊不是因为不能，而是因为不值得**：防作弊靠经济闭环而非技术封禁

## 十五、战斗系统

### 15.1 战斗界面布局

战斗不换场景——在当前界面展开。主内容区显示战斗文字描述，底部操作区切换为战斗按钮。

```
┌─────────────────────────────┐
│  你：气 85/120    HP ████░   │  ← 你和敌人的状态条
│  敌人：气 40/80   HP ██░░░   │
│─────────────────────────────│
│                             │
│  你催动《离火诀》，一道火舌     │  ← 文字战斗描述区
│  从掌心喷出——"焰返"！        │     双方行动以文字呈现
│  敌人被火焰吞噬，退了半步。    │
│  他的左臂在冒烟。            │
│                             │
│  敌人开始蓄力...（神识警告）   │  ← 红色预警提示
│─────────────────────────────│
│ [焰返 20气] [火环 35气]      │  ← 技能按钮（你装备的功法提供的特殊技）
│ [普攻 5气]  [防御]           │  ← 基础行动按钮
│ [丹药]     [逃跑]            │  ← 道具和策略按钮
└─────────────────────────────┘
```

### 15.2 核心机制：气的博弈

战斗是**回合制但不完全回合**——双方同时选择行动，然后一起结算。核心是**气**的管理：

```
气 = 你的行动资源
  • 上限取决于境界、体魄、功法
  • 回复速度取决于境界、灵脉浓度、丹药 buff
  • 所有行动消耗气：
    - 普攻：5 气（稳定的低消耗输出）
    - 技能：15-60 气不等（取决于功法）
    - 防御：10 气（抵消 50% 伤害）
    - 闪避：20 气（完全躲避，但连续使用消耗递增）
    - 爆发：全部剩余气（伤害 = 消耗气量 × 倍率）
  • 气不足时，可以扣血代替（1 血 = 2 气）
```

**每回合你需要决定**：这个回合全力输出？留气防御？还是在敌人蓄力时闪避？没有标准答案——取决于敌人的行动模式和你对他下一步的判断。

### 15.3 技能按钮

你装备的每本功法提供 2-3 个技能，显示为操作区的按钮。每个按钮上标注技能名和气消耗：

```
《离火诀》提供：
  [焰返 20气] — 闪避后回身刺击，附带火焰灼烧（3 回合持续伤害）
  [火环 35气] — 周身火焰爆发，弹开近身敌人并打断其当前行动
  [陨星 60气] — 大范围火陨，伤害极高但本回合无法防御

《玄水诀》提供：
  [水镜 15气] — 本回合格挡所有远程攻击并反弹 30% 伤害
  [暗流 40气] — 隐身 2 回合，下次攻击必定暴击
  [漩涡 50气] — 在敌人位置制造漩涡，他下回合无法闪避
```

**技能搭配**：战斗前你在洞府/坊市中装配功法（最多同时装备 3 本），进入战斗后这 3 本功法的技能按钮出现在操作区。提前想好搭配——火法 + 控制 + 防御？纯爆发玻璃大炮？这是战斗外的策略层。

### 15.4 神识预警

战斗中，如果你的神识高于敌人，你会在按钮区上方看到一行红色预警文字：

```
"敌人开始蓄力...（下回合可能释放强力技能）"
"敌人身形不稳...（下回合若受攻击则必定暴击）"
"敌人注意力分散...（下回合无法防御）"
```

神识不够 → 看不到预警 → 需要靠敌人的行为模式去猜。这是信息不对等造成的战术深度。

### 15.5 五行与灵根克制

每本功法有**五行属性**（金木水火土），每个角色有**先天灵根属性**（含变异灵根：风雷冰毒光暗磁虚空）。详细规则见 `SYSTEMS.md` 第二章。

战斗中属性克制不是简单的"伤害 ×1.5"——而是**战术效果的变化**：

| 克制关系 | 效果 |
|----------|------|
| 火克金 | 火系技能对金属性敌人额外削减其防御 buff |
| 金克木 | 金属性技能击中木属性敌人时，附加"破甲"——接下来 2 回合内该敌人受到的所有伤害 +20% |
| 水克火 | 水系技能命中火属性敌人时，消除其身上的灼烧/持续伤害 buff |
| 土克水 | 土系技能对水属性敌人附加"泥沼"——其闪避消耗翻倍 |
| 木克土 | 木系技能对土属性敌人附加"缠绕"——其下回合只能使用普攻 |

**变异灵根的克制规则**：
- 雷灵根（金+火）→ 被水克制（水克火），附带麻痹效果
- 冰灵根（水+金）→ 被土克制（土克水），附带冰冻效果
- 风灵根（木+火）→ 被金克制（金克木），闪避率天然 +10%
- 毒灵根（水+土）→ 被木克制（木克土），可叠加中毒层数
- 光/暗灵根 → 互相克制（伤害 ×1.5 双向），对基础五行伤害为基准值
- 虚空灵根 → 不参与克制链，对所有属性伤害均为基准值。只能修炼无属性功法

相生与灵根纯度效果：同系 buff 效果 +50%。灵根纯度越高，同属性功法伤害加成越大（天品 +50%）。相生属性攻击被生者时伤害 ×0.85，但为对方回复少量 HP。

### 15.6 敌人 AI

| 战术类型 | 行为模式 | 按钮提示 |
|----------|----------|----------|
| 狂攻型 | 连续使用高消耗技能，不留气防御 | 神识预警："敌人不顾防守，全力进攻" |
| 龟缩型 | 每回合留 50% 气用于防御 | 神识预警："敌人稳扎稳打，防守严密" |
| 诡诈型 | 假装气空引诱你爆发，然后闪避反击 | 神识预警不够高时，预警可能不出现 |
| 拼命型 | 残血时清空气槽释放最终技能 | 神识预警："敌人体内灵气剧烈波动——他在赌命" |

### 15.7 切磋与生死斗

| 模式 | 触发 | 界面变化 |
|------|------|----------|
| 切磋 | 对话中选"讨教" | 双方 HP 归零后自动回复，按钮区不显示[逃跑]，显示[认输] |
| 生死斗 | 遭遇敌对 NPC / 主动杀人 | [逃跑]按钮显示成功率（取决于你和敌人的速度差），失败则本回合无法行动 |

### 15.8 战斗节奏

- 一场对等战斗：8-15 回合，约 1-2 分钟
- 碾压战斗：2-3 回合，20 秒
- BOSS 战：15-30 回合，3-5 分钟
- 每回合你有 15 秒思考时间（可在设置中调整，也可关闭时限）

## 十六、天赋与根骨

### 16.1 角色创建时的随机天赋

开局时每个玩家随机获得 3 个天赋（从 30+ 天赋池中抽取），不满意可以用"道韵"重随一次。

| 天赋 | 效果 | 稀有度 |
|------|------|--------|
| 先天道体 | 修炼速度 +15% | 传说 |
| 火灵根 | 火系功法效果 +20%，水系 -10% | 稀有 |
| 剑心通明 | 剑法熟练度增长速度翻倍 | 稀有 |
| 丹毒免疫 | 不受丹毒 debuff 影响 | 传说 |
| 过目不忘 | 所有功法学习时间 -30% | 史诗 |
| 鸿运当头 | 奇遇触发概率翻倍 | 史诗 |
| 铁骨铮铮 | 炼体类功法效果 +25%，神识 -15% | 稀有 |
| 天生媚骨 | NPC 好感度获取 +50%，但同性 NPC 可能嫉妒 | 稀有 |
| 废灵根 | 修炼速度 -20%，但所有功法学习效率和效果不衰减（学再多也不打折扣） | 特殊 |
| 短命相 | 寿命上限 -30%，但战斗伤害 +20% | 诅咒 |

**设计意图**：天赋不是"谁更好"，而是"谁适合什么样的玩法"。废灵根看起来负面，但对什么都想学的人来说是最强天赋——你不需要在"专精一门"和"广泛涉猎"之间做选择。

### 16.2 后天觉醒

游戏中特定事件可以觉醒新天赋（替换或新增）：

- 渡劫成功 → 可能觉醒"雷劫淬体"
- 炼制出完美丹 → 可能觉醒"丹道天才"
- 被 NPC 背叛 → 可能觉醒"慧眼识人"
- 深渊探索中 → 可能觉醒或获得诅咒

天赋总数上限 = 5，满了之后需要遗忘旧天赋才能学新的。

## 十七、神识与探测

### 17.1 神识的核心作用

神识是**所有修士都可以使用的基础能力**：

| 神识用途 | 消耗 | 说明 |
|----------|------|------|
| 探测周围 | 持续消耗 | 显示小地图上的敌人/资源/NPC 位置 |
| 探查物品 | 一次性 | 鉴定未知道具、看穿伪装 |
| 识破 | 一次性 | 战斗中预判敌人下一次攻击方向（显示红色预警区域） |
| 威压 | 一次性 | 神识碾压弱者，低境界敌人获得恐惧 debuff |
| 搜魂 | 一次性 | 读取被击败 NPC 的记忆——获得功法残卷、秘境位置、门派秘密 |

### 17.2 神识的代价

**搜魂是最具争议的技能**：获得的信息价值极高，但每次使用积累"业力"。业力高 → 渡劫时因果劫更猛烈。被搜魂的 NPC 如果有师门/亲友，他们会知道——产生连锁仇恨。

**这是道德选择**：没有人禁止你用，但你承受得起后果吗？

### 17.3 神识屏蔽与反制

- 高阶修士可以感知到低阶修士的探测——并反向追踪
- 某些功法可以制造"神识迷雾"隐藏自身
- 特殊法宝可以伪造神识信号——让追踪者误判你的境界

## 十八、采集与生产玩法

### 18.1 采矿

采矿是**探索驱动的资源获取**，不是时机判断小游戏：

- 在探索中发现了矿脉 → 点击`[开采]`按钮
- 采矿结果取决于三个因素：**采矿技能等级 × 工具品级 × 矿脉本身品质**
- 不是每次都出完美矿——品质有随机范围（±20%），采矿技能越高范围越小
- 偶尔触发**"矿母"事件**：文字提示"你在矿石深处发现了一颗发光的矿母"，获得矿母（可带回洞府种植，再生矿脉）
- 高阶矿脉可能触发**"矿脉守护兽"事件** → 进入战斗

**操作按钮**：
- `[开采]` — 开始采矿（消耗体力，显示进度条 5-10 秒）
- `[深入]` — 往矿脉深处挖（风险更高，可能触发塌方或遇到守护兽，但产出更好）
- `[放弃]` — 离开矿脉

### 18.2 采药

采药是**时机 + 知识驱动的选择**，不是时机小游戏：

- 在探索中发现草药 → 点击查看其信息：名称、年份、当前品质、最佳采摘期倒计时
- 如果时机合适 → `[采摘]`，结果取决于采药技能 × 工具
- 如果时机不对 → 可以选择：
  - `[强行采摘]` — 品质打 6 折，但立刻到手
  - `[布禁制]` — 消耗一张符箓锁定这片药田，等时机到了再来（锁定期间其他 NPC 无法采摘）
  - `[标记]` — 在地图上标记位置，不锁定（别人也能采，但你不消耗符箓）
- 年份越高的草药，"最佳采摘窗口"越短——千年灵芝在游戏内 3 天后品质开始下降

**操作按钮**：
- `[采摘]` `[强行采摘]` `[布禁制]` `[标记]` `[离开]`

### 18.3 钓鱼

钓鱼不是时机小游戏——是**选择 + 等待**的休闲玩法：

- 在灵泉旁点击`[垂钓]`→ 开始钓鱼（显示进度条 10-30 秒）
- 进度条走完 → 文字提示"浮标猛然下沉！" → 点击`[提竿]`
- 钓上来的鱼有品级（凡品~仙品），取决于灵泉品质 × 钓鱼技能 × 运气
- 灵鱼用途：直接吃（临时 buff）、入药、**放入洞府灵泉继续养**
- 养到一定年份的灵鱼可能**化形成鱼妖**——可以收为灵兽
- 传说中有"龙门"秘境，灵鱼跃过龙门化龙

**操作按钮**：`[垂钓]` `[提竿]` `[放生]` `[收进鱼篓]`

### 18.4 种田

洞府药田不是装饰品。种田是炼丹师的**供应链管理**：

```
种田流程：
  选种（品级决定上限）
  → 播种（时辰影响发芽率）
  → 日常浇水/除虫（可用符箓自动化，也可以雇 NPC 药童）
  → 等待生长（年份越久品质越高）
  → 采收时机（如前所述）
  → 留种 or 入药
```

高阶炼丹师最终都会拥有自己的药田——因为交易行买不到 500 年份的草药，只能自己种。

## 十九、悬赏与江湖

### 19.1 NPC 悬赏系统

NPC 不是只有"好感度"这一个维度。整个修仙世界有一个**悬赏系统**：

- NPC 被欺负了（被抢、被杀亲友、被搜魂），会在坊市贴悬赏令
- 悬赏令上写明目标、赏金、是否要求生死斗
- 玩家可以接悬赏——猎杀 NPC 或猎杀其他玩家（如果接的是玩家悬赏）
- 完成悬赏获得灵石 + 赏金猎人名望

### 19.2 罪恶值与通缉

```
罪恶值来源：
  ├── 无故杀死正道 NPC (+50)
  ├── 搜魂 (+20)
  ├── 偷窃 (+10)
  ├── 修炼魔功被目击 (+30)
  └── 背弃师门 (+40)

罪恶值 > 100 → 正道宗门发布通缉令
罪恶值 > 300 → 太一道盟派出"执法队"（元婴期 NPC 组队追杀）
罪恶值 > 500 → 全服通告："魔头 XX 出现在 XX 区域，击杀者赏灵石 10 万"
```

**消罪方式**：
- 交灵石赎罪（太一道盟明码标价，贵到肉疼）
- 做善事（帮助 NPC、抵御兽潮）
- 时间流逝（每游戏年自然衰减 20 点）
- 转世重修（罪恶值清零，重新做人）

### 19.3 暗杀

高罪恶值玩家和顶级杀手之间产生了一个**玩法闭环**：

```
魔修玩家 A：罪恶值 500，悬赏金 10 万
杀手玩家 B：专接高额悬赏，追杀 A
A 可以选择：
  → 正面打（赢了悬赏金翻倍，输了……你知道的）
  → 躲起来等罪恶值衰减
  → 交灵石赎罪
  → 反过来悬赏 B（如果 B 也是高罪恶值）
```

没有 GM 介入，玩家之间的猎人-猎物关系自然形成。

## 二十、音乐与文艺

### 20.1 琴技

弹琴是一种技能，和炼丹、画符一样——你碰巧学会了，就会了。不归任何"流派"管，只是你会的众多本事之一。通过**"琴技"**这个独立技能树实现。

| 琴技 | 效果 |
|------|------|
| 清心曲 | 消除走火入魔风险 |
| 战歌 | 范围内的友方（含 NPC 队友）伤害 +15% |
| 安魂曲 | 降低妖兽的警觉范围 |
| 破阵曲 | 音波攻击，无视物理防御 |
| 迷魂曲 | 使低境界敌人陷入混乱 |
| 天问 | 终极曲目，直接对目标进行"道心拷问"——高概率使敌人陷入心魔 |

**学琴的方式**：找到隐世的琴师 NPC，完成一系列与战斗无关的任务（找到失传的琴谱、收集特定木材制琴、在特定地点弹奏等）——这是一种和"打打杀杀"完全不同的玩法节奏。

### 20.2 书道与画道

- **书道**：书法影响符箓的绘制精度。书法等级越高，画符成功率越高
- **画道**：作画可以"封印"天象——将一次七彩祥云画下来，变成一张"伪·七彩祥云图"，需要渡劫时使用
- **棋道**：下棋提升"神识"属性（最枯燥但最稳定的提升方式）。与特定 NPC 下赢可以获得其好感

### 20.3 茶道与酒道

与 NPC 社交的核心方式：

- **茶道**：请 NPC 喝茶 → 品茶过程中对话 → 好感度获取效率翻倍
- **酒道**：请 NPC 喝酒 → NPC 酒后吐真言（可能透露功法位置、秘境入口、宗门秘密）
- 不同的 NPC 偏好不同的茶/酒，送对口味事半功倍

## 二十一、体质与寿元

### 21.1 寿元系统

每个境界对应一个基础寿元上限。寿元耗尽 → 天人五衰开始（属性逐日下降直到死亡）。

突破大境界可以延长寿元。但这不是"逼你升级"——寿元的跨度很大，一个金丹期修士可以活 600 年，在游戏内时间流速下（现实 1 天 = 游戏 1 年），600 年约等于现实 1 年 8 个月。你有足够的时间慢慢来。

| 境界 | 基础寿元 | 阶段 |
|------|----------|------|
| 凡人 | 80 年 | — |
| 炼气 | 150 年 | 凡尘 |
| 筑基 | 300 年 | 凡尘 |
| 金丹 | 600 年 | 凡尘 |
| 元婴 | 1,200 年 | 凡尘 |
| 化神 | 3,000 年 | 凡尘 |
| 合体 | 8,000 年 | 凡尘 |
| 大乘 | 20,000 年 | 凡尘 |
| 渡劫 | 50,000 年 | 凡尘 |
| 地仙 | 100,000 年 | 登仙 |
| 天仙 | 300,000 年 | 登仙 |
| 真仙 | 1,000,000 年 | 登仙 |
| 玄仙 | 3,000,000 年 | 登仙 |
| 金仙 | 肉身不腐，不再自然衰老 | 登仙 |
| 太乙 | 同上 | 登仙 |
| 大罗 | 同上 | 登仙 |
| 混元 | 同上 | 登仙 |
| 鸿蒙 | 寿元不再有意义——你与天道同寿 | 道境 |
| 归墟 | 同上 | 道境 |
| 太初 | 同上 | 道境 |
| 造化 | 同上 | 道境 |
| 不朽 | 同上 | 道境 |
| 永恒 | 同上 | 道境 |
| 自在 | 同上 | 道境 |
| 逍遥 | 同上——你是一阵风，吹过就吹过了 | 道境 |

**延寿方式**：寿元丹（炼丹）、蟠桃（秘境产物）、某些禁术（吸别人的寿元——代价是业力暴增）。

**关于焦虑**：你不是在和时间赛跑。筑基期 300 年，金丹期 600 年——即使你什么都不做只是种田，也有大把的时间。突破不是为了"续命"，是因为你想看看下一个境界是什么样的。如果你不想看——那就不看。300 年够你种出世界上最老的桃树了。

### 21.2 天人五衰

寿命耗尽后不是立刻死，而是进入"天人五衰"debuff，逐阶段恶化：

```
第一阶段（剩余寿命 0 到 -10%）：全属性 -10%
第二阶段（剩余寿命 -10% 到 -30%）：全属性 -30%，无法修炼
第三阶段（剩余寿命 -30% 到 -50%）：全属性 -50%，炼丹/炼器/画符成功率减半
第四阶段（剩余寿命 -50% 到 -80%）：移动速度 -70%，战斗中被秒杀概率大增
第五阶段（剩余寿命 -80% 到 0）：躺在床上等死，需要别人喂延寿丹
```

**剧情价值**：你可能会遇到一个天人五衰第四阶段的 NPC 散修，他愿意把毕生所学传给你——只求一颗延寿丹。你有吗？你给吗？

## 二十二、江湖传闻与情报网

### 22.1 动态传闻系统

世界中的重大事件会自动生成"传闻"，在坊市公告板、茶馆 NPC 对话中传播：

```
传闻示例：
  "听说青云山附近最近频繁出现空间裂缝，有人从里面带出了仙品功法残卷。"
  "血煞宗长老在断魂崖被一个金丹期散修偷袭了，长老重伤，散修跑了。"
  "丹霞谷下个月要举办百年一度的丹道大会，头名奖励是一张道品丹方。"
  "有人在玄冰渊发现了一处未探索的上古遗迹，天机阁开价 5 万灵石买坐标。"
```

传闻的生成逻辑：
- 玩家的重大行为自动生成传闻（你杀了宗门长老 → 全服 NPC 都在议论）
- NPC 的重大行为也生成传闻（创造世界是"活的"的感觉）
- 部分传闻是真的，部分是 NPC 以讹传讹夸大的
- 天机阁卖的"情报"是验证过的传闻——这就是天机阁的价值

### 22.2 玩家情报网

玩家可以建立自己的**"情报网"**：

- 在坊市茶馆"收买"说书人 → 每天获得 3 条传闻
- 在宗门中安插"眼线" → 该宗门的人事变动、秘境发现第一时间知道
- 雇佣天机阁探员 → 追踪特定 NPC 或玩家的位置
- 建立"消息楼"（宗门建筑） → 被动收集所在碎片的所有传闻

情报本身就是商品——你知道某个秘境的位置但不想去？卖给天机阁换灵石。

## 二十三、法宝深度系统

### 23.1 法宝的类型

| 法宝类型 | 槽位 | 特点 |
|----------|------|------|
| 攻击法宝 | 法宝槽 1 | 飞剑、宝瓶、雷锤……自动攻击或主动释放 |
| 防御法宝 | 法宝槽 2 | 护盾、反弹、替身 |
| 辅助法宝 | 自由槽 | 炼丹加成、航行加速、神识增幅 |
| 本命法宝 | 专属槽 | 唯一，随境界成长，更换代价极高 |

### 23.2 法宝的灵性

法宝不是死物。长期使用的法宝会**产生灵性**：

```
灵性等级：
  Lv0 死物：无任何特效
  Lv1 通灵：在主人危险时自动触发一次防御
  Lv2 护主：可以自主判断何时使用（不消耗主人的气）
  Lv3 通人言：可以用文字和主人交流，提供战斗建议
  Lv4 化形：法宝可以变成人形/兽形，独立行动一段时间
  Lv5 自成一体：法宝拥有独立意志，可以自行修炼进化
```

灵性增长速度取决于：使用频率、主人境界、是否用灵魂碎片喂养。

**终极追求**：培养一件 Lv5 的法宝。它不只是装备——是你的第二个角色。

### 23.3 法宝的传承

本命法宝在主人死后不会消失，而是变成"无主法宝"。其他玩家可以继承，但灵性归零，且继承者需要使用"炼化"来解除前主人的印记——这个过程可能触发前主人的残魂反噬。

## 二十四、多人合作内容

### 24.1 区域 Boss（异步合作）

某些超级 Boss 的血量是**"全服共享"**的（服务器同步血量）：

```
示例：上古魔龙
  HP：10,000,000（全服共享）
  每个玩家单次最多打掉 5,000
  每次战斗后魔龙进入 30 分钟冷却（期间其他玩家不能挑战）
  击败后全服发放奖励（参与度越高，奖励越好）
  最后一击的玩家获得"屠龙者"称号
```

**设计意图**：不需要组队（保持半联机原则），但你和其他玩家是在"一起"打同一个目标。

### 24.2 世界事件（服务器驱动）

不依赖赛季的、更频繁的小型世界事件：

| 事件 | 频率 | 内容 |
|------|------|------|
| 坊市大集 | 每月一次 | 交易税减半，特殊 NPC 商人出现 |
| 兽潮来袭 | 随机 | 大量妖兽攻击坊市，全服玩家防守 |
| 灵脉喷涌 | 随机 | 某个碎片灵脉浓度 5x 持续 3 天，引发争夺 |
| 前辈讲道 | 每月一次 | 一个化神期 NPC 在坊市公开讲道，听讲的玩家获得修炼加速 buff |
| 秘境竞速 | 每周一次 | 一个限时秘境，全服玩家竞速通关排行榜 |

### 24.3 师徒系统（玩家间）

高境界玩家可以**收低境界玩家为徒**（不需要同时在线）：

- 师傅可以给徒弟寄送物品（通过交易行以"师门馈赠"特殊方式，免税）
- 师傅可以指定"修炼任务"给徒弟：完成某项后徒弟获得经验加成
- 徒弟突破大境界时，师傅获得"师德"——师德累计到一定程度可兑换特殊外观
- 一个师傅最多收 3 个徒弟，一个徒弟同时只能拜 1 个师傅
- 解除师徒关系需要双方同意或"逐出师门"（单方面，但有 7 天冷却）

## 二十五、转世

### 25.1 转世重修

如果你觉得这辈子走到头了——不是因为有什么"结局"在等你，只是你厌倦了当前的状态——可以选择**转世重修**：

- 保留所有功法知识（功法熟练度不重置）
- 境界归零，从头开始
- 转世后获得"宿慧"buff：修炼速度 +50% 直到回到前世境界
- 可以转世多次，每次获得不同的"轮回印记"——一些微小的永久被动

这不是"目标"，只是一个选择。就像一个手艺人某天决定关掉老店去另一个城市重新开始——带着所有经验，但没有行李。

### 25.2 轮回印记

| 轮回次数 | 印记名称 | 效果 |
|----------|----------|------|
| 第 1 次 | 宿慧 | 修炼速度 +50%（回到前世境界后消失） |
| 第 2 次 | 因果不昧 | 可以看到 NPC 的"善恶值" |
| 第 3 次 | 三世记忆 | 所有功法熟练度上限 +20% |
| 第 4 次 | 轮回之眼 | 可以看到隐藏的空间裂缝 |
| 第 5 次 | 彼岸印记 | 死亡惩罚减半 |
| 第 6 次 | 六道通 | 所有功法效率 +10%（可突破 100%） |
| 第 7 次 | 劫外身 | 渡劫成功率 +30% |
| 第 8 次 | 近道者 | 每本功法蜕变必出隐藏词条 |
| 第 9 次 | 轮回主 | 可以保留前 8 世中任意 3 个印记的效果 |

没有"轮回之主结局"。9 次转世之后，你只是——一个经历过九辈子的人。仅此而已。

游戏中有一个自然的时间流速（现实 1 天 = 游戏 1 个月），以下事件按游戏内时间循环：

| 每年九月九 | 重阳——渡劫成功率 +20% |

这些固定事件让玩家对"游戏内的日历"有感知，创造"等一个事件的期待感"。

## 二十七、更多细节玩法

### 27.1 储物戒管理

储物戒不是无限的。每枚储物戒有格子数上限（取决于品级），玩家需要管理背包：

- 材料 / 丹药 / 装备 / 符箓 / 杂物 各有专属页签
- 超出格子的物品可以"堆叠"（同种材料），但堆叠有上限
- 可以拥有多枚储物戒（切换使用），也可以炼制"乾坤袋"扩展
- 死亡时物品从当前装备的储物戒中随机掉落——所以有人会把重要物品放在"备用戒"里

### 27.2 改名与伪装

- 可以在天机阁购买"化名符"：交易行上架时使用假名（额外 2% 手续费）
- "易容丹"：短时间内改变外观，NPC 不会根据罪恶值认出你
- "匿息术"：隐藏自己的境界，让探测者误判

### 27.3 天气对战斗的影响

| 天气 | 战斗影响 |
|------|----------|
| 晴天 | 无影响（基准） |
| 雨天 | 火系技能伤害 -20%，水系 +20%，雷系 +50% |
| 雪天 | 冰系 +30%，移速 -10% |
| 雾天 | 远程技能命中率 -30%，神识探测范围减半 |
| 雷暴 | 随机落雷，无差别攻击场上所有人 |
| 沙尘 | 持续掉血，视野受限 |

### 27.4 残卷与复原

高阶功法往往以"残卷"形式掉落：

- 一本完整功法可能需要 3-9 页残卷拼合
- 残卷可以交易（每一页是独立物品）
- 不同玩家可能持有同一本功法的不同页 → 创造交易动机
- 天机阁可以付费查询"某本功法的第 N 页现在在哪个玩家手中"

### 27.5 镜像挑战

在坊市的"演武堂"中，点击`[镜像挑战]`按钮，系统生成**你的镜像**（和你完全相同的属性+装备+功法）：

- 进入战斗界面（同普通战斗 UI），对手是你的镜像
- 镜像使用你的功法但 AI 操控，可能使出你没想到的技能组合——"原来《离火诀》的焰返之后可以接《玄水诀》的暗流？"
- 战胜镜像 → 文字提示"你在战斗中灵光一闪" → 随机一本功法的熟练度大幅提升
- 每个境界只能挑战一次（16 个境界 = 最多 16 次）

## 二十八、突破与天劫

### 28.1 突破不是按钮，是一次事件

每个境界的第 9 层巅峰时，突破到下一个大境界不是点一个按钮。突破是一个**持续数日的进程**，期间你需要：

- **找个安全的地方**：洞府、宗门密室、或者偏僻荒野。突破期间你是脆弱的——灵气在你体内剧烈重组，你无法全力战斗
- **准备资源**：聚灵阵、护法丹药、灵石储备、防御法宝。准备得越充分，突破越平稳
- **面对天劫**：不是所有境界都会触发天劫，但以下节点必然触发：

| 境界突破 | 天劫类型 | 特点 |
|----------|----------|------|
| 筑基→金丹 | 三九雷劫 | 27 道天雷，最基础的天劫。布好阵法硬扛即可 |
| 元婴→化神 | 心魔劫 | 没有雷，但你会陷入幻境——看到最深的恐惧、最大的遗憾、最想回避的真相 |
| 大乘→渡劫 | 四九雷劫 | 36 道天雷，且天雷中掺杂心魔攻击 |
| 渡劫→地仙 | 因果劫 | 此生杀过的每一个人、每一个因你而受苦的生灵，都会在劫中以某种形式出现 |
| 金仙→太乙 | 时空劫 | 不是雷也不是心魔——是你被短暂地扔进一段错乱的时空中。你可能在劫中度过了一生那么长的时间，但现实只是一瞬。出来后你要面对那个"在劫中活了一辈子的自己" |
| 混元→鸿蒙 | 道心劫 | 天道直接"问你一个问题"。没有标准答案，但你的回答会永久改变你的某些属性 |
| 鸿蒙→归墟 | 寂灭劫 | 你的存在会短暂消失——不是死亡，是"不存在"。能否回来，取决于你对"存在"这件事的理解有多深 |
| 自在→逍遥 | 逍遥劫 | 没有雷、没有心魔、没有任何可见的劫难。只是——在那一个瞬间，你发现"你"这个概念的边界消失了。你不再是一个"人"、一个"修士"、一个"存在"。如果你还能回来——不是因为你的意志，只是因为一阵风吹过，碰巧把你带回来了 |

### 28.2 天劫的弹性

天劫的强度不是固定的，取决于：

- **业力**：搜魂、滥杀、背弃师门积累的业力会加成天劫强度
- **准备程度**：阵法、丹药可以减少天劫伤害
- **突破地点**：灵脉浓度越高，天劫越强——但突破后获得的加成也越大
- **旁人在场**：如果有人帮你护法（NPC 或玩家），天劫会认定"多人对抗"而增强
- **前世业力**：转世重修的人，天劫会算上前世的债

**设计意图**：天劫不是惩罚。一个业力深重的人面对更强的天劫，不是因为天道在审判他，只是因为——你搅动了太多因果，这些因果的涟漪叠加在一起，自然会在突破时产生更大的震荡。

### 28.3 突破中的选择

天劫期间可能出现**分支事件**：

```
心魔劫中，你看到一个场景：
  你年轻时杀过的一个散修，他的女儿站在你面前。
  她没有攻击你，只是看着你。
  
  你可以：
  A. 承认自己做过的事（心魔消散，但获得永久 debuff "愧疚"——对特定 NPC 类型的伤害 -5%）
  B. 否认——他该死（心魔加强，但获得永久 buff "铁石心肠"——心魔类攻击抗性 +10%）
  C. 尝试弥补——用自己的一部分修为去修复她破碎的灵根（掉一层小境界，但业力大幅减少）
```

突破不是"通过"就完了。你在突破中做的选择会留下永久的印记。

## 二十九、领域

### 29.1 什么是领域

境界达到太乙之后，你开始能在周身一定范围内**短暂地改写局部法则**。这个范围就是你的"领域"。

领域不是主动技能——它更像你的"气场"。你站在那里，周围的世界就开始按照你的规则运转。

### 29.2 领域的类型

领域的效果取决于你是"什么人"——不是你的选择，而是你过往行为自然沉淀的结果：

| 领域类型 | 触发条件（自然积累） | 效果 |
|----------|---------------------|------|
| 杀伐领域 | 战斗次数多、击杀多 | 领域内敌人持续受到威压，低境界者无法行动 |
| 生机领域 | 炼丹多、种田多、救人次数多 | 领域内友方持续回复，草药在领域内加速生长 |
| 幻象领域 | 使用符箓多、神识探查次数多 | 领域内敌人方向感紊乱，攻击有概率打偏 |
| 寂灭领域 | 独处时间长、社交少、在偏远碎片待得久 | 领域内所有声音消失，灵气流动减缓——敌人的技能施放时间翻倍 |
| 混沌领域 | 功法种类多而杂、多次转世 | 领域效果随机——每次战斗前摇一次，可能极强也可能完全没用 |

领域类型在太乙境界时**自动确定**，不由你选。它来自你的过往。

### 29.3 领域的成长

领域有大小和强度两个维度：

- **大小**：随境界提升自动扩大。太乙时半径约 3 步，归墟时可以覆盖一座小城
- **强度**：通过"领域内的经历"来提升。在杀伐领域内经历足够多的战斗 → 杀伐领域变强。在生机领域内种出足够多的高阶草药 → 生机领域变强

领域是你活出来的，不是你点出来的。

## 三十、道心与心魔

### 30.1 道心

**道心**是一个不可见的属性，从 0 到 100。它反映了"你对自己正在做的事有多确定"。

道心不决定你的强弱。一个道心值为 10 的人和一个道心值为 90 的人战力可以完全相同。道心影响的是：

- **心魔劫的难度**：道心低 → 心魔劫中幻境更真实更难摆脱
- **功法的蜕变概率**：道心高 → 功法熟练度满时触发隐藏词条的概率提升（从 10% 到最高 30%）
- **搜魂抵抗**：道心高 → 被搜魂时对方获得的信息更少
- **NPC 的感知**：某些高境界 NPC 能模糊感知你的道心值，影响他们对你的态度

### 30.2 道心的增长与衰减

道心不是靠"修炼"提升的，而是靠**"你做的事和你认为你应该做的事之间的一致性"**：

```
道心增长：
  + 你做了一个决定，然后坚持了它（即使是错误决定，坚持也有道心价值）
  + 你拒绝了某种诱惑（比如没趁人之危杀一个仇人）
  + 你在战斗中使出了一种新的连招组合——你对自身能力的理解加深了
  + 你在某个地方待了很久，对那片土地产生了归属感

道心衰减：
  - 你频繁切换行为模式——今天是商人，明天去杀人，后天躲进山里
  - 你背叛了某个 NPC 的信任
  - 你搜魂太多次（你在依赖别人的经验而非自己的判断）
  - 你转世重修（清零重置是道心的天然杀手）
```

### 30.3 心魔

心魔不是固定敌人，而是一个**随机的负面特质**，在以下时机出现：

- 第一次搜魂之后 → 可能获得心魔（概率 30%）
- 渡劫失败后 → 必定获得心魔
- 道心低于 30 时 → 每次大境界突破有 50% 概率获得心魔
- 杀死同门/NPC 师傅 → 必定获得心魔

心魔的例子：

| 心魔 | 效果 | 消除方式 |
|------|------|----------|
| 嗜血 | 战斗中伤害 +10%，但杀死敌人后必须再找一个目标，否则进入混乱 | 连续 100 天不战斗 |
| 疑心 | NPC 好感度获取 -50%，但你永远不会被 NPC 背叛 | 被一个 NPC 真诚帮助 3 次 |
| 贪婪 | 掉落率 +20%，但你无法丢弃任何物品——背包满了也不能扔 | 把自己炼的一件仙品以上物品送给陌生 NPC |
| 怯道 | 无法主动挑战比自己境界高的敌人 | 越级打赢一次战斗 |
| 执念 | 随机指定一个 NPC 为"执念对象"——你无法离开这个 NPC 所在碎片超过 10 天 | 和该 NPC 好感度达到 +80 或 -80 |

心魔不是纯负面。有些心魔有正面效果——代价是你不能无视它。

## 三十一、化身

### 31.1 什么是化身

境界达到金仙之后，你可以将一部分神识和灵气分离出去，形成一个**化身**。

化身不是分身——它不是你的复制品，而是一个**能力子集**的你。你可以把某几本功法"交给"化身，化身只能用这些功法战斗。化身的境界永远比你低一个大境界。

### 31.2 化身的用途

| 用途 | 说明 |
|------|------|
| 远程办事 | 让化身去某个碎片采集材料，而你的本体在另一个碎片修炼 |
| 守家 | 化身驻守洞府，来犯的 NPC 先过化身这一关 |
| 代你赴约 | 某个 NPC 约你在某地见面，但你不想去——派化身去 |
| 教学 | 化身可以收 NPC 徒弟，传授你交给它的那几本功法 |
| 试探 | 探索未知秘境？派化身先进去。化身被灭只是损失一部分神识（需要时间恢复） |

### 31.3 化身的限制

- 同时只能存在一个化身
- 化身不能交易（绑定本体）
- 化身不能突破境界
- 化身被消灭 → 本体神识受损，7 天内神识相关能力减半
- 化身的"人格"会逐渐偏离本体——如果化身在某地待了太久，它可能发展出自己的小习惯、口头禅。这些差异在收回化身时会作为一段模糊的"记忆碎片"回流到你的本体

### 31.4 化身的特殊事件

```
你的化身在一个偏远碎片待了三年（游戏内时间）。
当你收回它时，你收到一段记忆——

这三年里，化身帮助了一个 NPC 小女孩，教她识字、引气。
现在那个 NPC 已经筑基了。

你的功德列表里多了一条："在一个你看不到的角落，有人因为你而改变。"
```

你从不知道这件事。是你的化身自己做的。

## 三十二、世界塑造

### 32.1 这个世界是软的

大多数游戏的世界是硬的——山不能动，河不能改，你只是一个过客。这里不一样。

在天逆大陆，灵气浓郁到一定程度时，**物质本身就不再稳固**。一个足够强的人可以：

- **劈开一座山**：不是技能特效，是真的改变地形。劈开之后，山体裂谷会在那里，永久存在。后来的 NPC 会在裂谷边建房子，传闻会说"这是当年某人一剑劈出来的"
- **改道一条灵脉**：灵脉不是固定在地底下的——它们是流动的。你可以用阵法引导灵脉改道，把一个贫瘠的碎片变成灵地。当然，被抽走灵脉的地方会枯竭。那个碎片上住着的 NPC 会来找你
- **种出一片森林**：不是"建筑模式放树"，而是用生机类的功法 + 足够的时间 + 足够的耐心，在一片荒芜的碎片上催生出一整个生态
- **创造一块新碎片**：归墟境界的人可以从虚空中"捞"出混沌物质，捏成一块新的碎片。这块碎片有它自己的种子——会生成什么地形、什么资源、什么妖兽，部分由你决定，部分随机

### 32.2 改变的连锁反应

当你改变了世界，世界会回应你：

```
你劈开了一座山。
  → 山里的妖兽失去了栖息地，开始迁徙到附近碎片
  → 附近坊市的妖兽材料价格下跌（供应暴增）
  → 一个 NPC 采药人在裂谷底部发现了一条新的矿脉
  → 矿脉的消息传到铸剑城，铸剑城派了人来勘探
  → 你无意中引发了铸剑城和当地宗门之间的资源争夺
```

你不是在"做任务"。你只是在某个下午想试试自己的剑气有多锋利——然后世界因此发生了一系列你完全没有预料到的事。

### 32.3 没有"建造模式"

改造世界没有 UI 面板，没有网格对齐，没有"确定/取消"。你想劈山，就真的走到山前面，蓄满气，砍下去。砍成什么样取决于你的力度、角度、和当时用的功法。

**这意味着改造是笨拙的、不可逆的、有意外后果的。** 就像真实世界一样。

## 三十三、因果网络

### 33.1 看得见的因果

境界达到大罗之后，你可以"看到"因果——不是模糊的直觉，而是**可视化的线条**：

```
激活因果视野后，你看到的画面：
  
  你面前的 NPC 身上延伸出数百条半透明的线——
  红线是他伤害过的人
  蓝线是他帮助过的人  
  金线是他欠了恩情的人
  黑线是因他而死的人

  每一条线都延伸到虚空中，指向线的另一端。
  如果你顺着一条线走，你能找到那个人。
```

### 33.2 因果的可操作性

看到因果之后，你可以：

- **追溯**：顺着一条因果线找到源头——"那个偷我丹药的 NPC 现在在哪？"顺着你和他之间的因果线走
- **切断**：消耗大量道韵，切断你和某个人之间的因果。从此你们互相找不到、记不住、任何因果上的联系都消失。代价是那段因果中的所有记忆——好的坏的——全部模糊
- **嫁接**：把一条因果线从你身上转移到你的化身上——让仇人追杀你的化身而非本体
- **强化**：主动加深和某个 NPC 的因果绑定——以后他突破时你可以感知到，他危险时你会收到预警

### 33.3 因果的代价

看到因果是一个诅咒，不是一个 buff。大多数大罗境界的人选择不去看——因为看到自己身上有多少条黑线（因你而死的人），不是一个让人舒服的体验。

**你可以选择不激活因果视野。** 这是你的自由。有些真相不值得知道。

## 三十四、禁忌知识

### 34.1 有些东西不该学

这个世界的功法不是"等级越高越安全"。有些知识是**禁忌**——

不是因为有人禁止，而是因为这些知识本身会改变知道它的人。

| 禁忌知识 | 获取方式 | 代价 |
|----------|----------|------|
| 天道的碎片真相 | 在归墟碎片中找到残片 | 知道之后，你无法再享受任何"重复性"的快乐——种田、钓鱼、炼丹这些日常开始变得苍白 |
| 真名 | 从远古遗迹中的石碑上读到 | 你知道了自己的"真名"之后，别人用真名叫你，你无法拒绝——这是比任何功法都强的控制 |
| 虚空的语言 | 在虚空深处待足够久，自然学会 | 学会后你可以和虚空中的存在交流。但它们也会和你交流——你无法关掉这个频道 |
| 死亡的真相 | 死过一次并自己复活（非替身符） | 你对死亡不再恐惧。但你也对活着不再有期待——两个方向都失去了重量 |
| 情道 | 和一个 NPC 达到 +100 好感后，那个 NPC 自然死亡 | 你会知道什么是"失去"。战斗中获得 buff"不为所动"——控制类技能对你无效。但你无法再和任何 NPC 建立 +50 以上的好感 |

### 34.2 禁忌知识的传播

禁忌知识可以被**传播**：

- 你可以在坊市茶馆里把"真名"的知识讲给一个陌生人听
- 他听到了，他也被影响了
- 他可能会恨你，也可能会来找你追问更多
- 没有人阻止你传播。后果你自己承担

## 三十五、虚空深处

### 35.1 规则之外的地方

已知的碎片——无论多偏远——都遵循基本规则：有灵气、有重力、有时间流逝、生物需要呼吸。

但虚空深处不是这样。在远离所有已知碎片的虚空深处，你可能会遇到：

| 异常区域 | 规则变化 |
|----------|----------|
| 静默区 | 所有声音消失。不是"安静"，是波形本身不传播。你在里面喊叫，连你自己的耳朵都听不到。符箓无法激活（需要声波触发）。在里面待太久的人会开始自言自语——出来后也改不掉 |
| 逆流区 | 时间倒流。你走进去，看到自己倒退着走出去。如果你在里面战斗，你会在战斗结束后才感受到战斗开始时的伤口。炼丹在这里是不可能的——你不知道草药是往前长还是往后长 |
| 虚无法则区 | 灵气不存在。你的所有功法都失效。你只是一个普通人——直到你走出来的那一刻，灵气重新涌入你的身体，那种感觉像溺水后吸到的第一口气 |
| 镜像区 | 里面有一个"你"。不是敌人，不是复制品。就是一个和你一模一样的存在，站在那里，看着你。你做什么它就做什么。如果你待太久，你会开始分不清哪个是你 |
| 巨兽坟场 | 虚空中漂浮着巨大生物的骸骨。每具骸骨都像一座城市那么大。它们是活的还是死的？没有人知道。只是偶尔——非常偶尔——其中一具骸骨，会睁开一只眼睛 |

### 35.2 虚空深处没有"探索度"

没有成就，没有探索列表，没有"虚空深处探索度 37%"。

你去那些地方，只是因为你想去。你发现了什么，你可以选择告诉别人，也可以选择不告诉。你死在某个异常区里，没有人会知道。

## 三十六、时间的重量

### 36.1 物品会记住

一件物品不只是它的属性。一件物品**经历过什么**——它会记住。

| 物品经历 | 获得的效果 |
|----------|-----------|
| 佩戴了 100 年（游戏内时间） | 自动获得词条"故旧"——装备者受到致命伤害时，物品替主人挡一次（物品碎裂） |
| 用它杀了 1000 个敌人 | 获得词条"千人斩"——对同类型敌人的伤害 +5%，但 NPC 靠近时会感到不安 |
| 从师傅传给徒弟 | 获得词条"师承"——徒弟佩戴时，有概率在战斗中"灵光一闪"——使出一招师傅曾经用过的技能 |
| 在同一个洞府中放置了 50 年 | 获得词条"安居"——洞府防御阵法效果 +10% |
| 被偷走过 | 获得词条"失而复得"——无法再被同一人偷走，但原主人佩戴时会在战斗中听到若有若无的叹息 |
| 主人死在这件装备旁边 | 获得词条"遗志"——下一个主人会做一个梦，梦中看到前任主人的最后时刻 |

### 36.2 地名是人给的

碎片没有名字。地图上的地名是**玩家或 NPC 起的**，叫的人多了就固定下来。

- 你发现一片碎片，在上面第一个建了洞府。你给这片碎片起名叫"桃花源"。以后所有经过这里的 NPC 和玩家，在地图上看到的名字就是"桃花源"
- 如果你起的名字太难听，别人可以不用——他们叫它"那个破地方"。如果叫"那个破地方"的人比叫"桃花源"的人多，地图会更新
- 一个地方如果发生了一件大事，名字可能会变。比如"青云山"某个 NPC 在那里渡劫失败，后人开始叫它"坠仙崖"

### 36.3 古老本身就是力量

一件凡品的铁剑，如果存在了 10000 年（游戏内时间），它就不再是凡品。

不是因为它被附魔了——只是因为**时间本身会在物品上积累重量**。一个万年老物件，即使没有任何附灵，也会自然拥有一些微弱的特殊效果——比如握在手里时，你会隐约看到这把剑曾经见过的所有主人。

## 三十七、流浪与定居

### 37.1 两种活法，没有哪个更好

你可以选择**流浪**——没有固定洞府，走到哪算哪。碎虚舟就是你的家。

你可以选择**定居**——在某块碎片上建洞府，种田，养灵兽，几十年不挪窝。

两种活法都有自然的得失，不是"奖励"，是因果：

| | 流浪者 | 定居者 |
|------|--------|--------|
| 探索发现 | 更容易发现稀有碎片和秘境 | 对外界的感知逐渐降低 |
| 战斗应变 | 适应力强——在不同环境中战斗无惩罚 | 在洞府周边战斗有额外 buff |
| NPC 关系 | 交到很多朋友，但都是浅交 | 交到的朋友少，但可以很深 |
| 物品积累 | 无法囤积大量物品（船空间有限） | 仓库可以无限大 |
| 技能倾向 | 更适合学很多种功法但都不精通 | 更适合专精少数功法 |
| 自然事件 | 你不在时，你关心的 NPC 可能出事 | 你在时，可以及时介入 |

### 37.2 随时可以切换

没有"你选了流浪就不能建洞府"——你今天在某个碎片觉得舒服，就可以建个洞府住下来。住了十年觉得无聊了，可以锁上门，回到船上，继续流浪。

洞府不会消失。你回来时它还在——可能是原样，可能被 NPC 占了，可能门前的桃树已经长成了桃林。

## 三十八、传承——你留下什么

### 38.1 功法是可以写出来的

你学会的任何功法，都可以**抄写一份**。

- 抄写需要时间、笔墨（朱砂+灵纸）、以及你对这本功法的熟练度至少 60%
- 抄本和原本效果完全相同（功法内容不因抄写而改变）
- 你可以把抄本送给 NPC、卖给交易行、埋在某个遗迹里等有缘人发现

### 38.2 自创功法

当两本功法的熟练度都达到 100%，你可以尝试**融合**它们，创造一本新功法。

融合的结果不是可预测的——取决于两本功法的性质、你的境界、你的道心值、以及一点点运气。可能会产生一本更强的功法，也可能产生一本废功。

自创的功法你可以**命名**。这个名字会被永久记录在功法上。如果抄本流传出去，被其他人学到——你的名字作为"创始者"会留在功法描述里。

### 38.3 你死后

你死了之后：

- 你的本命法宝变成无主之物，等待下一个有缘人
- 你的洞府变成遗迹（如果你的洞府够老够有名，可能成为"XX 散人的洞府"被后人探索）
- 你自创的功法如果流传出去了，会继续存在——NPC 和玩家会继续学它、抄它、融合它。你在功法描述里的名字是永久的
- 你的化身如果当时在外面执行任务，不会消失——它会变成一个独立的 NPC，带着对你的模糊记忆继续活着

如果你转世重修了，你可能会在某个坊市遇到一个 NPC——它是你上一世的化身。它还认得你。但你已经不是它记忆中的那个人了。

## 三十九、夺舍

### 39.1 你不是非得死

死亡不是终点——如果你不愿意接受的话。

当一个修士肉身被毁但神识未散时，有一个选择：**夺舍**。找到另一个活着的人，侵入他的身体，碾碎他的神识，占据他的肉身。

夺舍的条件：
- 目标境界必须**远低于**你生前的境界。一个化神期的人只能夺舍筑基以下的肉身。越强的神识需要越强的容器——元婴以上几乎无法被夺舍，除非夺舍者比你强两个大境界
- 目标必须**意志薄弱**：道心低、重伤中、或者正在渡劫/突破时被偷袭
- 夺舍不是暗中的事——你进入目标身体后，你们两个的神识会在识海中**面对面搏斗**。你赢了，身体归你。你输了，你的神识被对方吸收——他获得了你的一部分记忆和能力

### 39.2 夺舍的代价

夺舍不是免费的"重生"。你付出的是：

- **境界跌落**：夺舍成功后，你的境界会掉一个大境界。化神夺舍成功 → 元婴
- **灵根不兼容**：新身体有自己的灵根属性。你生前是火系功法大师，夺舍到一个水系灵根的身体里——你的火系功法威力永久打七折
- **业力暴增**：夺舍是天道法则中最重的罪行之一。夺舍一次，业力 +300。这意味着下次渡劫时，因果劫的强度会翻倍
- **原主的残魂**：夺舍成功后，原主的残魂不会完全消失。它会在你神识深处，偶尔在梦里出现。如果你道心低，残魂可能反噬
- **没有人信任夺舍者**：NPC 可以通过神识感知到你"不是原来的你"。被你夺舍的目标如果有亲友——他们会知道。他们会来找你

### 39.3 你也会被夺舍

夺舍不是只有你能做的事。道德对它没有豁免权。

- 一个比你高两个大境界的 NPC 如果肉身被毁，他可能盯上你
- 夺舍者的神识进入你的识海时，你会看到他的记忆碎片——他生前是什么人、为什么死、为什么选了你。这是战斗前的"前奏"——你可以在这些记忆碎片中寻找他的弱点
- 如果你在识海搏斗中打赢了夺舍者，你反过来吸收了他的神识。你会获得他的一部分功法知识、一些他不知道你知道了的秘密、以及——一条新的因果线：他生前所有的仇人，现在把你当成他

### 39.4 夺舍链

一个被夺舍过的人，如果再被夺舍——第三个夺舍者会在识海中同时面对两个残魂：原主和第一个夺舍者。这可以一直叠加下去。

```
某个倒霉的筑基期 NPC 体内有五个残魂在打架。
他自己是第六个——他还没死，只是被压制了。
这六个人在识海里组成了一个微型的"社会"。
```

你可以找到这样的 NPC。你可以选择帮他清理掉其他的残魂，还他自由。也可以选择成为第七个。

## 四十、暗杀与密谋

### 40.1 地下的坊市

太一道盟管辖的坊市是"光明"的——有规则、有税、有公告板。

但每个坊市下面都有一个**黑市**。它不在任何地图上标注。入口是一个 NPC 的口令、一块特定的墙砖、或者在某个时辰站在某棵树下。

黑市里没有规则。没有税。没有交易记录。没有鉴定。一切全凭眼力。

黑市提供的东西：

| 服务 | 说明 |
|------|------|
| 悬赏令 | 不是坊市公告板上那种"通缉魔头"的官方悬赏。是**"我要某人死"**的那种。你出灵石，有人接。目标是 NPC 还是玩家——都一样 |
| 情报买卖 | 天机阁卖的是公开情报。黑市卖的是**"某个人的功法弱点""某个宗门的护山大阵破绽""某个洞府的防御漏洞"** |
| 禁术交易 | 血祭、摄魂、炼尸——这些功法在正常坊市是买不到的。在这里可以 |
| 假身份 | 一个新的道号、一份伪造的身份玉牌。如果你想换个身份重新开始——或者逃避追杀 |
| 销赃 | 杀人夺宝拿到的东西。交易行可能会识别出"此物品与近期某凶案的失物描述相似"（概率性），黑市没有任何审查——但黑市的买家也都不是善茬 |

### 40.2 暗杀网络

黑市的悬赏令系统是一个**去中心化的暗杀网络**：

- 你发布悬赏 → 灵石存入"中间人"（一个黑市的 NPC，他的信誉是黑市唯一的规则——如果他私吞灵石，整个黑市的人都会追杀他）
- 有人接了悬赏 → 灵石冻结
- 目标死亡（服务器验证） → 灵石划给杀手
- 目标还活着 → 悬赏继续挂着，第一个人放弃的话第二个人可以接

**没有人知道杀手是谁。** 包括悬赏发布者。

### 40.3 密谋

比暗杀更复杂的是**密谋**。密谋不是一次性交易——是一系列的行动，针对同一个目标，由多个互不认识的人各自完成自己的环节。

```
一个完整的密谋：

玩家 A 想搞垮某个 NPC 宗门。
  → A 去黑市发布"密谋"——目标：玉华宗
  → 玩家 B 接了"情报"环节——去偷玉华宗的护山大阵阵图
  → 玩家 C 接了"内应"环节——拜入玉华宗，成为内门弟子，获取信任
  → 玩家 D 接了"执行"环节——在 B 和 C 完成后，攻入玉华宗
  → A 全程不知道 B、C、D 是谁。B、C、D 之间也不知道彼此
  → 如果任何一个环节失败、或者有人背叛——整个密谋崩溃
```

密谋的吸引力：**你不需要自己动手。** 你有灵石、有仇人、但不想染血——把灵石放在黑市，自然有人帮你处理。

密谋的风险：**你不知道接你密谋的人是谁。** 可能是你的仇人。可能是那个 NPC 宗门的弟子。可能是一个专门钓鱼密谋发布者然后去告密的"白道猎人"。

## 四十一、禁术

### 41.1 超出正常功法范畴的东西

禁术不是"更强的功法"。禁术是**以他人为代价换取力量**。

| 禁术 | 效果 | 代价 |
|------|------|------|
| 血祭 | 杀死一个 NPC，将其修为转化为自身灵气。转化率取决于 NPC 境界——杀一个金丹可以抵十年修炼 | 业力 +100，被杀 NPC 的因果线全部转移到你身上——他的仇人现在恨你，他的亲友现在找你。而且血祭过的灵气是"脏"的——渡劫时天雷会对脏灵气产生额外的反应 |
| 摄魂 | 不完全杀死一个 NPC——只是抽走他的七情六欲中的一部分。被摄魂的 NPC 不会死，但会变成"活死人"——有呼吸有心跳，但眼睛里没有光了 | 摄魂者的神识中会永久残留被摄魂者的"情感碎片"。你会在随机时刻感受到那个 NPC 的恐惧、悲哀、或者——他对某个人的爱。你感受到他对妻子的爱，但你从未见过那个女人 |
| 炼尸 | 将死去的 NPC 尸体炼制成傀儡。傀儡保留生前境界的 70%，绝对服从。可以帮你战斗、采矿、守家 | 每个傀儡占用你的一部分神识（上限 = 你的神识总量 / 每个傀儡的消耗）。傀儡太多 → 你自己的神识被稀释 → 本体变弱。而且 NPC 看见你带着一具尸体走路，他们对你的好感度自动 -50 |
| 吞灵 | 吞噬另一个修士的灵根。目标灵根被毁，你的灵根获得其属性的一部分 | 被吞噬的灵根在你体内不是"融合"，是"共存"。火灵根的你吞了一个水灵根——你的体内会有一个"水火战场"。修炼时随机触发内伤 |
| 诅咒 | 消耗自身寿命，对目标施加一个永久 debuff。比如消耗 50 年寿命，让目标的下一次渡劫必定失败 | 诅咒不是远程魔法——你需要目标的一件贴身物品、一根头发、或者一滴血。而且诅咒反弹：如果目标渡过了那次天劫（概率虽小但非零），诅咒会加倍反弹到你身上 |

### 41.2 禁术的获取

禁术不会在正常渠道出现。你需要：

- 在黑市购买（来源不明，可能是某人从远古遗迹中带出来的）
- 在虚空深处发现（某些异常区中，正常法则失效，禁术的"代价"也可能失效——但这不可预测）
- 被 NPC 传授（某个走投无路的老修士，临死前把禁术传给你——"拿着吧，也许有一天你会需要"）
- 自己研究出来（当一个功法熟练度极高的人陷入极度愤怒或绝望时，可能"顿悟"出该功法的禁术变体）

### 41.3 使用禁术不意味着你是反派

一个从来没有杀过人的炼丹师，在自己的道侣被 NPC 杀死后，第一次走进黑市，买了一份诅咒的材料清单。

她不是反派。她只是一个伤心的人。

**这个系统不审判你。** 它只是把工具放在那里。你用不用、怎么用、用完之后怎么面对自己——那是你的事。

## 四十二、背叛

### 42.1 信任是脆弱的

在这个世界里，信任不是 UI 上的一个数字。信任是**你把自己的一部分暴露给别人**——然后期待他不会利用它。

可以被背叛的关系：

| 关系 | 背叛方式 | 后果 |
|------|----------|------|
| 师徒 | 师傅偷学徒弟自创的功法后逐出师门 | 徒弟获得心魔"师恩如刀"，此后对所有师徒关系 +50% 戒心 |
| 道侣 | 一方在另一方渡劫时不护法反而偷袭 | 被偷袭方如果活下来，获得永久 buff"不再信人"——此后双修效率归零，但心魔抗性翻倍 |
| 道友 | 组队探索秘境时，一方抢了宝物后把另一方推进陷阱 | 被背叛方如果生还，自动获得"识人"能力——可以看到 NPC 和玩家是否曾经背叛过别人 |
| 合伙人 | 在坊市共同经营商号的两个人，一方卷款跑路 | 跑路方获得灵石，但黑市会多出一条新的悬赏令——发布者是那个被卷走毕生积蓄的人。悬赏令不用灵石支付，用"背叛者的一只手" |
| 宗门 | 弟子窃取宗门秘典后叛逃 | 宗门发布追杀令——不是官方的，是黑市的。追杀令上写着"不论死活"。但弟子窃取的秘典已经抄了三份，埋在了三个不同的碎片上 |

### 42.2 背叛不是"剧情"

背叛不是系统安排的事件。背叛是**玩家或 NPC 自主做出的选择**。

NPC 会根据性格、处境、和你的关系历史来判断是否背叛你。一个道心很高的 NPC 几乎不会背叛。一个被逼到绝路的 NPC——比如他需要一颗救命丹药而你手里有——可能会为了那颗丹药背叛你们十年的道友关系。

你也是。你也可以背叛任何人。

### 42.3 背叛的连锁

一次背叛不只是两个人的事。

```
A 背叛了 B。
  → B 的朋友 C 和 D 开始追查 A。
  → A 逃到另一个碎片，换了个假身份。
  → C 在黑市悬赏 A 的位置。
  → 一个不相干的 E 接了悬赏，找到了 A，但没有杀他——E 把 A 的位置卖给了一个更恨 A 的人：F。
  → F 是 A 十年前背叛过的另一个人。F 一直在等这一天。
```

一个背叛可以发酵十年。一个十年前种下的因，十年后在一个你完全不认识的杀手身上结成果。

## 四十三、审讯与酷刑

### 43.1 信息不在数据库里

有些信息不在交易行、不在天机阁、不在功法描述里。有些信息只存在于**某个 NPC 的脑子里**。

你可以问他。他可以选择不回答。

然后你可以选择——

| 手段 | 效果 | 代价 |
|------|------|------|
| 利诱 | 给他灵石、丹药、功法、或者帮他做一件事 | 没有代价。但有些 NPC 不吃这一套 |
| 威逼 | 威胁他——威胁他的宗门、他的家人、他在意的东西 | 即使成功，他对你的好感度永久 -100。他会记住。他以后变强了会来找你 |
| 搜魂 | 直接暴力读取记忆 | 业力 +20，被搜魂者神识受损（可能永久降低境界上限），搜到的信息是碎片化的——你可能得到关键情报的 30%，剩下的需要拼图 |
| 熬炼 | 不是搜魂的暴力读取——是一种更"精细"的方式。用神识侵入对方识海，耐心地、一层一层地揭开他的记忆。熬炼需要数小时甚至数天 | 比搜魂更完整——可以精确提取特定信息。但熬炼对施术者自己也是一种折磨——你会完整地、共情式地体验对方记忆中的恐惧和痛苦。道心低于 50 的人熬炼一次可能自己先崩溃 |

### 43.2 熬炼的双向性

熬炼的时候，你们两个是连接在一起的。他感受到你的神识侵入——你感受到他的痛苦。

```
你熬炼一个 NPC，试图找到他藏功法的地方。

你进入他的记忆——看到他年轻时在一个秘境中找到了那本功法。你很接近了。

但同时你也在体验他那一天的恐惧——秘境塌了，他的师弟被困在石头下面。
他选择了带着功法逃出去，师弟死了。

你现在知道两件事：
  1. 功法藏在哪（你要的信息）
  2. 这个 NPC 三十年来每天晚上都梦到师弟的脸（你不想要的信息，但你知道了）
```

熬炼完之后，你们两个都变了。他是被熬炼的人。你是熬炼的人。但某种程度上，你也熬炼了你自己。

## 四十四、奴役

### 44.1 你可以拥有别人

通过禁术"锁魂"，你可以将一个 NPC 变成你的**奴仆**。

奴仆不是灵兽、不是傀儡、不是弟子。奴仆是**人**——会思考、有情感、有自己的想法。但他无法违背你的命令。你让他杀他的朋友，他会哭，但他会杀。

锁魂的条件：
- 目标境界必须低于你
- 目标必须处于无法抵抗的状态（重伤、昏迷、或者自愿——有些人为了换取家人的安全，自愿成为奴仆）
- 锁魂之后，你和奴仆之间建立了一条**单向因果线**：你感知他的位置、他的情绪、他的生死。他感知不到你

### 44.2 拥有奴仆的代价

锁魂不是免费的 buff：

- 每一个奴仆占用你**神识上限的固定比例**。奴仆太多 → 你自己变弱
- 奴仆会恨你。他的恨意不会消失——只是被压制了。如果你神识受损（比如化身被灭），压制力减弱，奴仆可能反噬
- NPC 看到你带着奴仆，会根据他们的道德观做出反应。正道 NPC 好感度暴跌。血煞宗的人会觉得你"手法不错"
- 你的道心会受到影响。具体取决于你自己的道德感——如果你觉得"没问题"，道心不受影响。如果你内心深处觉得"这样做不对"但你还是做了——道心缓慢下降

### 44.3 释放奴仆

你可以随时释放奴仆。释放之后：

- 他对你的恨变成了"曾经恨过"
- 他的境界上限永久降低了——锁魂造成的损伤是不可逆的
- 他不会感谢你释放了他。你只是停止了伤害，你没有"救"他
- 因果线断裂，你不再能感知他。但如果多年后你们再次相遇——他会认出你

## 四十五、情报黑市

### 45.1 每个人的秘密都有价格

黑市的另一层是**情报黑市**。天机阁卖的是"客观信息"——秘境坐标、功法位置、矿脉分布。

情报黑市卖的是**"人的信息"**。

| 情报类型 | 示例 | 来源 |
|----------|------|------|
| 弱点情报 | 某个高手的功法破绽、战斗中会暴露的漏洞 | 可能是他的同门卖出来的，可能是被他打败过的人分析出来的 |
| 位置情报 | 某个人目前的藏身之处 | 情报贩子有自己的追踪网络 |
| 把柄 | 某个人做过的不可告人的事 | 可能是目击者卖的，可能是熬炼出来的 |
| 人际关系图 | 某个人在乎谁、恨谁、欠谁、被谁欠 | 长期跟踪收集的 |
| 身份情报 | 某个"散修"的真实身份——他其实是被灭门的某某宗门的唯一幸存者 | 来源不明，可能是他本人放出来的假情报 |

### 45.2 情报的真假

情报黑市不保证情报是真的。一个情报贩子卖给你的"弱点情报"可能是目标本人故意散布的假消息——你按照假弱点去打他，反而正中他的陷阱。

验证情报真伪的方式：
- 多买几份——不同贩子对同一个目标的描述如果一致，可信度提高
- 去天机阁交叉验证——天机阁的情报是真的，但不卖"人的弱点"这种灰色信息
- 自己去试探——但这会暴露你的意图

### 45.3 你的情报也在那里

这是一个令人不舒服的事实：**情报黑市里也有你的档案。**

你不知道是谁卖的、卖了什么、卖给了谁。某个你完全不认识的人可能正在看你的功法破绽分析，而这份分析是你十年前交过手的一个 NPC 写的——他甚至记得你出第三剑时左脚会稍微往前多挪半步。

你可以花钱买回自己的情报——情报贩子会把你的档案下架。但下架不等于销毁。他可能明天又上架了。

你也可以追查是谁卖了你的情报。这将是一个完全独立的故事线。

## 四十六、执念

### 46.1 人不是理性动物

NPC 不是。玩家也不是。

**执念**是一个 NPC 或玩家在特定条件下产生的**非理性驱动力**。它不是任务、不是目标、不是 buff——它是"我忍不住要这样做"。

执念的产生条件：

| 触发事件 | 产生的执念 | 表现 |
|----------|-----------|------|
| 被夺走一件珍爱的物品 | "寻回" | NPC 会放弃手头的一切，追踪那件物品的下落。追踪可能持续数年。如果你在交易行买了一件"有执念附着"的物品——你不知道，但有一个 NPC 正在来找你的路上 |
| 亲眼目睹挚友/道侣被杀 | "复仇" | 所有其他行为暂停。NPC 开始疯狂修炼、搜集凶手情报、或者直接去凶手可能出现的地方蹲守。如果凶手是你——你会频繁地在不同碎片"偶遇"这个 NPC |
| 在一个地方待了超过 50 年 | "守护" | NPC 无法离开那个地方超过 30 天。如果有外人侵入，他会不计代价地驱逐。这个 NPC 可能只是一个筑基期的老人——但他在自己守护的土地上，战力翻倍 |
| 反复被同一个人打败（3 次以上） | "超越" | NPC 以打败那个人为唯一目标。不是仇恨——是执念。他会研究那个人的招式、情报黑市里买那个人的弱点、闭关修炼针对性的功法 |
| 得到一件远超自己境界的物品 | "占有" | NPC 无法丢弃或交易这件物品——即使它对他没用。如果有人想夺走它，他会拼命。如果物品被偷，触发"寻回"执念 |
| 被救了一命 | "报答" | NPC 会在恩人最危险的时刻出现——他不知道恩人有危险，但他就是去了。这不是系统安排的，是执念驱动的"巧合" |

### 46.2 玩家的执念

玩家也可以产生执念。当你反复做某件事、或者经历了某个极端事件后：

```
你连续 10 次去同一个秘境，只为了刷一本功法残卷。
第 10 次失败后，你获得执念"不得到它不罢休"。
  → 你在那个秘境中的战斗能力 +20%
  → 但你在其他所有地方的经验获取 -30%（你的心思不在这里）
  → 得到那本功法后，执念解除
  → 如果一直得不到——执念可能转化为心魔
```

玩家的执念不是惩罚。它只是**承认你已经在做的事**——然后放大它。

### 46.3 执念是可利用的

一个情报贩子发现一个 NPC 有"守护"执念——他知道这个 NPC 永远不会离开某个碎片。他把这个情报卖给了一个想躲仇家的人。那个人在那个碎片上住了十年，仇人找不到他——因为他没有离开过，而 NPC 守护者的存在让所有外来者都很显眼。

你不是在利用 bug。你只是在利用人性。

## 四十七、记忆市场

### 47.1 记忆是可以取出来的

境界达到化神之后，神识足够精细，你可以把一段**记忆提取出来**——不是复制，是**剪切**。

提取出来的记忆变成一件物品：一颗**忆珠**。任何人捏碎这颗忆珠，都会完整地体验那段记忆——包括记忆中的情感、感官、和当时的心境。

一颗忆珠包含的可以是：

| 内容 | 价值 |
|------|------|
| 一本功法的完整修炼过程 | 比功法更珍贵——你不只学到功法，你还体验到创始人是怎么"悟到"这本功法的 |
| 一个秘境的完整探索经历 | 你可以"提前"体验一次秘境——知道哪里有陷阱、哪里是死路。但你也会体验到第一次探索者的恐惧和兴奋 |
| 一段感情 | 有人把和已故道侣的回忆提取出来——"我不想忘记，但太痛了。先存着吧" |
| 一个秘密 | 某个宗门的阴谋、某个碎片的隐藏入口。你捏碎忆珠，你知道了他知道的事——但你也体验到了他得知这个秘密时的震惊和恐惧 |
| 一段技能 | 一个铁匠把他打铁 50 年的手感提取出来。你捏碎忆珠，你的双手突然"知道"怎么打铁了——虽然你的大脑还不太理解 |

### 47.2 忆珠的交易

忆珠在正常坊市是**禁止交易**的。太一道盟的理由是："记忆不应该成为商品。"

但在黑市，忆珠交易是最活跃的市场之一：

- 一个即将渡劫的修士愿意花全部积蓄买一段"成功渡劫的经验"——某个渡过同类天劫的人的记忆
- 一个凡人想体验"飞在天上是什么感觉"——他买了一段炼气期修士的飞行记忆
- 一个想学炼丹的人买了三段不同炼丹师的"成丹瞬间"记忆——然后自己去尝试，成功率大幅提升

### 47.3 忆珠的风险

你捏碎忆珠，你体验了一段不属于你的记忆。那个记忆中的情感会**残留**在你身上：

- 你买了一个陌生人的战斗记忆。你现在知道怎么用剑了。但你也在梦里反复梦到一个你从未见过的女人——那是记忆原主人的道侣。他爱她。你现在也对她有一种模糊的、没有理由的好感
- 你买了一段"秘境探索"的记忆。记忆原主人在那个秘境中死了一个队友。你现在进入那个秘境时，会莫名其妙地感到悲伤——即使你从未带过队友
- 你买了太多忆珠。你开始不太确定哪些记忆是你自己的，哪些是买来的。你的道心缓慢下降——**"我是谁"这个问题的答案变得越来越模糊**

### 47.4 被提取记忆的人

一个人提取出一段记忆后，那段记忆就**没有了**——不是模糊了，是消失了。像从来没有发生过一样。

一个修士把他道侣死后那段最痛苦的日子提取出来，卖给了忆珠贩子。他现在不痛了。但他也**不记得那段日子里的任何事了**——包括那段日子里他领悟到的一本功法的蜕变。那些也一起被提走了。

有些人专门靠"卖自己的记忆"维生。他们把高价值的体验（渡劫、功法顿悟、奇遇）提取出来卖掉，然后用灵石换修炼资源。他们是忆珠市场的内容创作者。

他们不记得自己卖掉了什么。他们只知道钱包变鼓了，脑子里有一些空白——不像是忘了什么，是像那里从来就没有过东西。

## 四十八、誓言

### 48.1 话说出口，就不能收回了

在天逆大陆，誓言不是修辞。在灵气浓度足够高的地方——或者双方境界足够高——**誓言会产生实际的约束力**。

誓言的发生条件：
- 双方自愿（被迫的不算——胁迫下说的誓言不会产生约束）
- 双方都是修士（凡人的誓言没有约束力——这是不公平的，但这是世界的规则）
- 誓言内容具体（"我会帮你"太模糊，没有约束。"如果下个月十五之前你拿来三株龙涎草，我就把《离火诀》传授给你"——具体，会产生约束）

### 48.2 违背誓言的代价

违背一个产生了约束力的誓言，代价取决于誓言的内容和双方的境界差：

| 违背类型 | 代价 |
|----------|------|
| 小事违约 | 道心 -5。天劫时心魔劫中会重现那个违约的场景——对方站在那里，问"你答应过我的" |
| 大事违约 | 道心 -20，业力 +50。一条新的因果线——"失信线"——连接你和被你辜负的人。这条线可以被因果视野看到。看到的人会知道——你是一个会违背誓言的人 |
| 生死契约违约 | 道心 -50，业力 +200。天道对你进行"标记"——此后所有誓言（即使你真心想遵守）的约束力自动减弱，因为天道不再完全相信你的话 |
| 血誓违约 | 血誓是最高等级的誓言——双方滴血为盟。违背血誓的人，下一次渡劫时天道会"加料"——不是更难的雷劫，而是一个**镜像**。你的对手是你自己——不是心魔，是一个完全复制了你的属性和功法的实体。它只有一句台词："你自己都不信自己。我来替你信。" |

### 48.3 誓言的妙用

誓言不只是一种限制——它是一种**信用的基础设施**：

- 两个互不信任的玩家想在黑市做一笔大交易。他们在天道的见证下起誓——"货到付款，违约者天道标记"。这笔交易不需要中间人
- 一个玩家想向一个 NPC 证明自己的诚意。他主动提出血誓——"我保证不伤害你的宗门。除非你的宗门先伤害我。" NPC 看到血誓成立，好感度直接 +30
- 一方想设陷阱：在誓言中留了模糊的措辞。"我会给你'足够'的灵石"——什么是"足够"？如果双方对誓言的解释不一致，天道以**誓言的文字而非意图**为准。所以誓言中每个字都可能是武器

## 四十九、名声

### 49.1 名声不是善恶值

善恶值是一个数字——你杀了多少人、做了多少善事。名声不一样。名声是**别人怎么看你**。

名声有多个维度，每个维度独立计算：

| 名声维度 | 高意味着 | 低意味着 | 如何传播 |
|----------|----------|----------|----------|
| 守信 | 你说到做到 | 你的话不值钱 | 与你交易过的 NPC 向他们的社交圈传播 |
| 勇武 | 你很强，而且你敢打 | 你懦弱或者很弱 | 战斗的目击者传播 |
| 慷慨 | 你舍得给 | 你抠门 | 接受过你馈赠的 NPC 传播 |
| 残忍 | 你下手狠 | 你仁慈（也可能是软弱） | 幸存者和目击者传播 |
| 博学 | 你懂很多 | 你见识短浅 | 与你交流过的 NPC 传播 |
| 孤高 | 你很少与人来往 | 你交游广阔 | 有 NPC 想找你但找不到时传播 |

### 49.2 名声是滞后的、扭曲的

一个 NPC 在某个碎片上听到你的名声时，他听到的可能是一年前的版本，而且经过了三个人的转述。

```
你的真实行为：你在青云山杀了一个欺负凡人的恶霸。

第一手传播（目击者 NPC）："有个修士在青云山杀了王麻子。"
第二手传播（听到的人转述）："青云山有个狠人，当街杀了个修士。"
第三手传播："听说青云山出了个杀人狂，路过的人他都杀。"

三个月后，一个完全不了解你的 NPC 在另一个碎片上：
"你就是那个青云山杀人狂吧？我听说你连路过的都不放过。"
```

你无法控制自己的名声。你可以尝试——去坊市茶馆里讲自己的故事、贿赂说书人传颂你的善举——但这本身就是一种行为，会产生新的名声。"这个人到处说自己做了好事"——博学维度微增，慷慨维度微妙地下降（因为真正的慷慨不需要宣扬）。

### 49.3 利用名声

名声是一把双刃剑：

- "残忍"名声高：NPC 在你面前更顺从，但更少有人愿意和你深入交往。情报黑市里关于"如何打败你"的情报需求会上升——因为有人怕你，怕你的人想提前做准备
- "守信"名声高：你可以在没有中间人的情况下借到灵石。黑市里的 NPC 贩子会主动找你合作——"你这个人说话算话，我有个生意想找你一起做"
- "孤高"名声高：你越少出现在人群中，NPC 对你的想象越大。你从一个"不太社交的人"变成了"神秘高人"。有人开始编造关于你的传说。有些传说甚至是夸你的——只是完全不属实

## 五十、气运

### 50.1 运气是存在的，但它不是随机的

每个生灵出生时都有一个**气运值**——你可以理解为"天道对你的关注度"。这个值在 0 到 100 之间浮动。

气运不决定你的命运。它决定的是**"偶然事件发生在你身上的频率和倾向"**：

- 气运高 → 探索时更容易遇到奇遇、炼丹时更容易出隐藏效果、突破时天劫强度略微降低
- 气运低 → 探索时更容易遇到麻烦、修炼时更容易走火入魔、在坊市容易被小偷盯上

气运不是你做了什么——是天道的目光在你身上多停留了一秒。

### 50.2 气运是可以转移的

气运不是固定的。它会自然变化——也**可以被操纵**：

| 方式 | 效果 |
|------|------|
| 在一个地方长期居住、日久生情 | 气运缓慢上升（天道喜欢"扎根"的人） |
| 频繁换碎片、居无定所 | 气运缓慢下降（天道还没来得及注意到你，你已经走了） |
| 帮助很多人（不是做善事——是被很多人真心感谢） | 气运上升。每一次真心的感谢，都是在天道的账簿上给你加了一笔 |
| 被很多人恨 | 气运下降。恨意也是关注——但是负面的 |
| 夺走别人的气运 | 有一种禁术可以**偷走**别人的气运。但被偷者的厄运也会一起过来——你偷了一个气运 90 的人的好运，也偷了他年轻时埋下的某个因果炸弹 |
| 把气运存在物品里 | 你可以把一部分气运"寄存"在一件物品上。那件物品放在洞府里，洞府周围的好事增多——灵草长得更快，妖兽不敢靠近。但你自己在外的气运降低 |

### 50.3 气运的感知

金仙以上可以模糊感知自己和他人的气运——不是数值，是**"感觉"**：

- 气运高的人身上有一种"光"——不是很亮，是那种"今天天气真好"的舒适感
- 气运低的人身边会发生微小的倒霉事——茶杯莫名打翻、灵石不小心掉进下水道、飞剑总是落在一坨鸟粪旁边
- 如果你看到一个气运极低的人——你可能想帮他，也可能想远离他。他的厄运不会传染给你，但你站在他旁边时，你会感到一种隐约的、说不清楚的压抑

## 五十一、梦境

### 51.1 梦不是假的

修士的梦不是普通的梦。当你的神识在睡眠中脱离肉身的束缚，它会进入一个**共享的领域**——梦境世界。

梦境世界不是另一个地图。它是**所有正在做梦的修士的神识共同构成的一个暂时的、流动的空间**。

在梦境中：
- 你可能会"漂"进一个陌生人的梦里——看到他的记忆碎片、他的恐惧、他的秘密
- 你自己的梦也可能被闯入——某个你不认识的人在梦里看到你年轻时做过的某件事
- 两个在现实中从未见过面的修士，可能在梦里吵了一架。醒来后他们不记得对方的脸，但记得那种"我讨厌那个人"的感觉

### 51.2 入梦

化神以上可以主动**入梦**——不是自然睡觉时被动漂进去，是有意识地进入梦境世界并导航。

入梦可以用来：

| 用途 | 说明 |
|------|------|
| 托梦 | 给某个特定的人传递信息。不保证他醒来后记得——梦里的信息容易被扭曲。一个清醒时"我需要你帮我找一株草药"的请求，在梦里可能变成一片会说话的森林 |
| 窥梦 | 进入某个人的梦，观察他的记忆碎片。不是搜魂的暴力读取——是安静的旁观。他梦到自己的童年，你站在旁边看着。这不算犯罪——因为没有伤害。但如果你被发现了，他会记住你的"感觉"。下次在现实中遇见你时，他会觉得你"很眼熟，但不知道为什么" |
| 噩梦植入 | 在某个人的梦里埋下一个恐惧。不是控制他的行为——只是让他每次想起某件事时，心里多了一层阴影。常用于报复——不需要见面，不需要流血 |
| 梦中授业 | 在梦里教一个人功法。学习效率比清醒时低 50%，但可以在对方不知情的情况下进行——他以为是自己"顿悟"了 |
| 梦杀 | 在梦境中攻击一个人的神识。如果在梦中对神识造成了足够的伤害，他醒来时会头痛欲裂、神识受损。梦杀最难的部分不是杀伤——是**找到他的梦**。梦境世界无穷大，你要在一个无数人梦境交织的汪洋中找到一个人的梦，就像大海捞针 |

### 51.3 清醒梦

有些修士选择**不睡觉**。长期不睡觉 → 神识得不到休息 → 疲劳累积 → 道心缓慢下降。

有些修士选择**永远在梦里**。他们服用某种丹药，肉身进入假死状态，神识永远留在梦境世界。他们在现实中是"失踪人口"，在梦里是游荡的鬼魂。如果你在梦境中遇到一个比周围更"清晰"的人——他可能就是永久居民。

## 五十二、疯狂

### 52.1 理智不是默认状态

每个修士都有一个**理智值**。它不是血条、不是蓝条——是**"你对自己和世界的认知有多稳固"**。

理智值从 0（完全疯狂）到 100（完全清醒）。正常人是 100。大多数修士在 70-100 之间浮动。

理智下降的原因：

| 原因 | 下降幅度 |
|------|----------|
| 搜魂太多次 | 每次 -5。你看了太多别人的记忆——你开始不太确定哪些记忆是你的 |
| 熬炼别人 | 每次 -10。你共情式地体验了对方的痛苦——那不是你的痛苦，但你的神识不在乎 |
| 买了太多忆珠 | 每颗 -3。太多拼凑的记忆 = 拼凑的自我 |
| 在虚空深处待太久 | 每天 -1。法则失效的地方，自我认知也会失效 |
| 被夺舍未遂 | -20。有人试图把你从你自己的脑子里赶出去——即使你赢了，那种"我不确定我还是不是我"的感觉会持续很久 |
| 禁术使用超过 3 次 | 每次 -15。以他人为代价的力量——你的潜意识知道这不自然 |
| 道心低于 20 很久 | 持续缓慢下降。你不知道自己在做什么，所以你也不太确定自己是谁 |

### 52.2 疯狂的形态

理智降到不同阈值时，表现不同：

| 理智值 | 表现 |
|--------|------|
| 70-100 | 正常 |
| 50-70 | 轻微的强迫行为——你必须每天做某件特定的事（比如泡一壶茶、打一套剑法），不做就焦躁。你是正常的，但有了"仪式感" |
| 30-50 | 你开始看到不存在的东西——不是幻象攻击，是眼角余光里的影子、功法的文字在你读完后还在动、灵石上的纹路在对你眨眼。你知道那不是真的。但你还是看到了 |
| 10-30 | 你和你自己的对话变成了两个人的对话。你在战斗中会"听到"一个声音建议你做某个操作——那个声音是你自己。你开始用"我们"称呼自己 |
| 0-10 | 你是清醒的——比任何时候都清醒。你看到了世界真正的样子。别人看不到。他们说你疯了。但你只是——看到了。这个阶段的人会做出不可预测的事。可能是一剑劈开坊市。可能是坐在山巅沉默了十年。可能是在地上挖了一个很深的洞，一直往下挖——"我要看看下面是什么" |

### 52.3 恢复理智

理智不是永久损伤。它可以恢复：

- 在一个安全的地方长时间静养（洞府最佳）
- 和道心高的 NPC 交谈——他们的"确定"会感染你
- 种田、钓鱼、泡茶——反复的、可预期的、温和的日常
- 提取一段痛苦的记忆（制成忆珠，卖掉或销毁）——去掉源头
- 转世重修——理智重置为 100。但轮回印记中可能多一条："你曾经疯过。你知道那是什么感觉。你知道怎么再疯一次。"

## 五十三、伪造

### 53.1 一切都可以伪造

这是一个没有物品指纹的单机世界。任何东西都可以被模仿——能不能被识破，取决于买家的眼力。

| 伪造类型 | 方法 | 效果 |
|----------|------|------|
| 赝品装备 | 炼器时刻意模仿某件名器的外观和属性分布。外观、品级几乎一样，但内在属性是假的——看起来是仙品，用起来是灵品 | 在交易行上架时，交易行有概率识破赝品（识破率取决于交易行的鉴定等级和赝品的制作精度）。识破了 → 标注"疑似赝品"。没识破 → 正常上架，买家自求多福。黑市不鉴定，全凭买家眼力 |
| 假忆珠 | 把一段虚构的记忆——从未发生过的事——制成一颗忆珠。捏碎它的人会"回忆"起一件从未发生过的事 | 可以用来骗人——"这颗忆珠里有某个宗门的功法"→ 没有。但你捏碎后会有一段"学习功法"的虚假记忆，你可能会以为自己学会了——直到你试了一下，什么都没发生 |
| 假身份玉牌 | 手艺够好的炼器师可以伪造身份玉牌。换了玉牌 = 换了一个身份。天机阁的数据库里你还是你——但在 NPC 面前，你是另一个人 | 用来逃避追杀、潜入宗门、或者重新开始。但如果你用假身份和 NPC 建立了真实的关系——当假身份被揭穿时，那个 NPC 受到的伤害是真实的 |
| 假功法 | 把一本功法的内容稍作修改——改掉某个关键步骤——然后装订成真功法的样子。学的人可能会走火入魔 | 卖给仇人、放在秘境里当陷阱、或者在黑市当"真功法"卖 |

### 53.2 伪造者的名声

一个出名的伪造者——他伪造的名器在外观上甚至比真品更精美——在黑市是一个受人尊敬的职业。

他不做骗子。他只做"复制品"。买的人知道是复制品。收藏家喜欢——"我买不起真品，但这件赝品已经够好了。"

但总有人把他的赝品当成真品去骗下一个人。他不负责"别人怎么用"。他只负责造。

## 五十四、冥婚与死后世界

### 54.1 死后不是一片空白

一个 NPC 死了之后，他的神识如果没有被夺舍、没有被炼成傀儡、没有被吞噬——它会去哪里？

它会进入**冥域**。一个只有在死后才能到达的碎片。活人进不去——除非你死过一次又复活。

冥域里有什么——没有人知道。但有一些东西从冥域**回来**了：

- **遗言**：某些 NPC 死后，他们的遗言会以梦境的形式出现在他们在乎的人梦中。不是幽灵——是残余的神识在最后一次触碰这个世界
- **冥器**：在冥域中浸染过的器物。它们看起来和正常物器一样——但它们触碰活人皮肤时，活人会感到一阵冰凉的悲伤。不是温度，是情感
- **回魂**：极少见的情况下，一个死去的 NPC 会在某个特定的日子（他的忌日、或者他生前最爱的人的生日）短暂地回到阳间。他不能说话、不能触碰东西。他只是站在角落里，看着。然后消失

### 54.2 冥婚

如果一个修士的道侣死了——而他不愿意接受这个事实——他可以举行**冥婚**。

冥婚仪式：
- 需要道侣的遗物（生前最珍爱的物品）
- 需要道侣生前待过最久的地方（通常是洞府）
- 需要一滴活人的血——和一张写了道侣名字的黄纸

仪式完成后：
- 活人和死者的名字在天道的婚约簿上被绑定
- 活人获得一个永久 buff："未亡人"——防御力 +20%（为了活着等到重逢），但双修效率永远为 0（因为你已经结婚了）
- 死者的残魂在冥域中获得力量——因为被一个人以这样的方式铭记着，他在冥域不再是孤魂
- 活人有时会在梦里见到死者。不是噩梦植入、不是幻觉——是真的他。只是短暂地、模糊地

如果后来这个活人又爱上了别人——天道婚约不会自动解除。他需要去冥域入口，烧掉当年的婚书，说一声"对不起"。然后冥婚解除，buff 消失，道心 -10。不是因为天道在惩罚他——是因为他自己放不下，道心自己跌了。

---

> 这个世界的核心问题是：**没有人告诉你该做什么，但你的每一个选择都会留下痕迹。**
