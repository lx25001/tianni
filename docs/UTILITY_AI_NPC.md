## 一、核心理念

### 1.1 NPC 即玩家镜像

效用 AI 的核心原则：**NPC 的决策模型与玩家的决策空间完全对称**。玩家能做什么，NPC 就能做什么——唯一区别是 NPC 由 AI 评分驱动，玩家由自由意志驱动。

```
玩家能做的事：
  修炼 → 挂机积累修为 → 突破
  探索 → 移动房间 → 发现资源/NPC/秘境
  战斗 → 攻击妖兽/NPC → 掉落
  交易 → 坊市买卖
  建造 → 洞府升级
  社交 → 送礼/切磋/道侣

NPC 同样能做：
  修炼 → 后台积累修为 → 突破（概率 × 道心 × 环境）
  探索 → 移动房间 → 采集资源/遭遇事件
  战斗 → 与妖兽/NPC 战斗 → 胜负由属性 + 概率决定
  交易 → 与坊市/NPC 交易
  建造 → 拥有洞府 → 升级灵脉
  社交 → 与玩家/NPC 建立因果
```

### 1.2 不做行为树，做效用评分

行为树是「if A then do B」。效用 AI 是「在 N 个选项中，选当前最想做的」。

```
行为树（我们不做的）：
  饿了 → 吃饭
  被攻击 → 逃跑

效用 AI（我们要做的）：
  当前有 8 个可选行动
  每个行动都有一个「欲望分数」
  分数最高的行动胜出
  
  修炼: 42 分
  采集: 18 分
  交易: 35 分
  战斗: 8 分
  ……
  → 修炼分数最高 → 本回合修炼
```

## 二、需求维度

每个 NPC 有 6 根「需求条」，随时间推移自动增减，类似《模拟人生》的需求系统。需求条的值直接影响对应行动的评分。

| 需求 | 满值 | 自然衰减 | 影响 |
|------|------|----------|------|
| **生存** | 100 | -1/时辰 | HP 过低时急剧上升。触发炼丹/购买丹药/逃跑行为 |
| **修为** | 100 | -0.5/时辰 | 每突破一层后归零（满足感），缓慢回升。触发修炼行为 |
| **财富** | 100 | -2/天 | 拥有灵石 < 100 时快速上升。触发交易/采集/出售行为 |
| **安全** | 100 | -1/天 | 在危险区域/被仇家追杀时急剧上升。触发移动/阵法/逃跑 |
| **社交** | 100 | -3/天 | 长时间不与人交互时上升。触发拜访/送礼/切磋 |
| **突破** | 100 | 仅在 9 层满时激活 | 追求大境界突破的执念。触发吞噬天材地宝/悟道/因果 |

**需求驱动的行动选择**：

```
如果 生存需求 = 90（快死了）→ 疗伤/逃跑的效用评分 +50
如果 修为需求 = 85（很久没修炼了）→ 修炼的效用评分 +40
如果 财富需求 = 70（灵石快花完了）→ 采集/交易的效用评分 +30
如果 社交需求 = 60（很久没见人了）→ 拜访 NPC 的效用评分 +20
```

## 三、评分模型

### 3.1 公式

```
行动效用评分 = 基础偏好 × (1 + 需求匹配度) × 性格系数 × 环境因子 × 风险评估
```

### 3.2 各因子说明

**基础偏好**（每个行动有自己的基础分，模拟 NPC 的职业/身份偏好）：

| 行动 | 炼丹师 | 剑修 | 散修 | 宗门长老 |
|------|--------|------|------|----------|
| 修炼 | 50 | 60 | 40 | 30 |
| 炼丹 | 80 | 10 | 20 | 25 |
| 战斗 | 10 | 70 | 30 | 25 |
| 采集 | 40 | 15 | 50 | 15 |
| 交易 | 30 | 15 | 40 | 20 |
| 社交 | 15 | 10 | 20 | 50 |

**需求匹配度** = 对应需求条当前值 / 100。如果生存需求 = 90，所有涉及生存的行动（疗伤、买药、逃跑）匹配度 = 0.9。

**性格系数**（见第五章性格调节）。

**环境因子**：身处的房间有灵气节点？修炼 +20%。有矿脉？采集 +30%。有坊市？交易 +15%。深夜子时且野外？安全需求 +40%。

**风险评估**：目标如果是对手，实力对比 = 目标战力 / 自身战力。比值 > 1.5 → 该行动的「安全」权重降低，风险评估拉低总分。

### 3.3 最终决策

```
1. 枚举当前可用的所有行动（8-15 个）
2. 为每个行动计算效用评分
3. 选出最高分
4. 如果最高分 < 阈值（10 分）→ 本回合发呆/闲逛
5. 如果最高分 ≥ 阈值 → 执行该行动
6. 执行后需求条更新 → 下一回合重新评分
```

## 四、行动池

每个 NPC 根据其境界、位置、身份，拥有不同的可选行动。行动池不是固定的——境界越高，可选行动越多。

### 4.1 全 NPC 通用行动

| 行动 | 耗时 | 需求影响 | 前提 |
|------|------|----------|------|
| 原地修炼 | 1 时辰 | 修为 -20（缓解修炼需求） | 无 |
| 移动（去相邻房间） | 1 时辰 | 无 | 无 |
| 原地采集 | 1 时辰 | 财富 -10 | 所在房间有资源 |
| 使用丹药 | 即时 | 生存 -30~50 | 背包有丹药 |
| 发呆 | 1 时辰 | 无 | 所有行动评分都低于阈值时触发 |

### 4.2 境界解锁行动

| 行动 | 解锁境界 | 说明 |
|------|----------|------|
| 坊市交易 | 炼气 1 层 | 买卖物品 |
| 挑战妖兽 | 炼气 3 层 | 攻击房间内的妖兽 |
| 拜师 | 炼气 5 层 | 向高境界 NPC 请求成为弟子 |
| 炼丹 | 筑基 1 层 | 消耗材料制作丹药 |
| 建造洞府 | 金丹 1 层 | 在地图空房间开辟洞府 |
| 收弟子 | 金丹 5 层 | 接受低境界 NPC 的拜师请求 |
| 创建宗门 | 元婴 1 层 | 消耗大量灵石创建宗门 |
| 阵法布设 | 化神 1 层 | 在洞府布置防御阵法 |
| 跨界探索 | 大乘 1 层 | 进入虚空裂缝探索 |
| 因果推演 | 金仙 1 层 | 主动探查自身的因果线 |
| 自创功法 | 太乙 1 层 | 融合已学功法创造新功法 |

### 4.3 事件触发行动

| 行动 | 触发条件 | 说明 |
|------|----------|------|
| 复仇 | 宿敌出现 | 战斗效用 +100（强制高分），几乎必定触发 |
| 逃命 | HP < 20% | 移动行动的效用 ×3，向安全区域撤离 |
| 抢救 | 道侣濒死 | 放弃当前行动，冲向道侣所在房间 |
| 争夺 | 目睹稀有资源 | 采集/交易效用 +80 |
| 警觉 | 陌生人进入洞府范围 | 安全需求 +50 |

## 五、性格调节

每个 NPC 在创建时随机生成性格参数，影响所有行动的效用评分。性格不是「这个 NPC 是什么样的人」——性格是「这个 NPC 在面对选择时，天平会往哪边倾斜」。

### 5.1 五维性格

| 性格维度 | 范围 | 高值倾向 | 低值倾向 |
|----------|------|----------|----------|
| **胆量** | 0-100 | 主动挑战高境界对手，高风险行动 +30% 分 | 逃避冲突，逃跑阈值降低 |
| **野心** | 0-100 | 为了突破可以牺牲财富/社交（突破需求 ×1.5） | 安于现状，突破需求增长慢 |
| **勤勉** | 0-100 | 修炼/采集/炼丹等生产行动 +20% 分 | 发呆/闲逛概率增加 |
| **忠诚** | 0-100 | 不容易背叛宗门/道侣，复仇概率翻倍 | 可以为了利益出卖任何人 |
| **贪婪** | 0-100 | 交易时出价更极端，采集欲望更强 | 对财富无所谓 |

### 5.2 性格之间的交互

```
高野心 + 低忠诚 = 为了突破不择手段（可能杀道侣、盗宗门秘典、血祭路人）
  这类 NPC 是世界的「不稳定因素」——他们制造最多的因果事件

高胆量 + 高野心 = 最可能跨越大境界的 NPC
  但也是最容易陨落的 NPC——他们挑战的对手往往比自己强

低胆量 + 高勤勉 = 稳扎稳打的「乌龟型」NPC
  修炼慢但死得慢，往往能靠寿元优势熬到高境界

高忠诚 + 低野心 = 理想的弟子/道侣
  不会背叛，但也不会主动追求突破——需要外力推
```

### 5.3 性格的演化

性格不是固定的。重大事件可以改变性格参数：

```
被道侣背叛 → 忠诚 -30，胆量 +10（不再轻信，但更不怕死）
弟子陨落 → 勤勉 +20，野心 -10（化悲痛为修炼，但对突破失去兴趣）
获得奇遇 → 野心 +15（看到了更大的可能性）
被仇家击败 → 胆量 -20 或 +30（崩溃 or 觉醒，取决于道心）
```

## 六、境界分层

不同境界的 NPC 有不同的「决策重点」和「时间尺度」。一个筑基期的 NPC 和一个玄仙期的 NPC，对同一个事件「万妖之乱」的反应完全不同。

| 境界段 | 决策重心 | 时间尺度 | 典型行为 |
|--------|----------|----------|----------|
| 炼气-筑基 | 生存 + 积累 | 游戏天 | 采集、修炼、躲避妖兽、找师傅 |
| 金丹-元婴 | 突破 + 建府 | 游戏月 | 建造洞府、炼丹、收弟子、宗门任务 |
| 化神-大乘 | 探索 + 因果 | 游戏年 | 虚空探索、经营宗门、寻找奇遇 |
| 地仙-金仙 | 悟道 + 自创 | 游戏十年 | 法则领悟、自创功法、因果经营 |
| 太乙-永恒 | 超脱 | 游戏百年 | 已不在乎日常琐事。只在「天道异变」级别的事件中出手 |

## 七、与因果网络联动

效用 AI 不是孤立运行的——它需要通过因果网络感知世界状态。

### 7.1 因果感知

```
NPC 在评分时，会查询自己的因果网络子图：

如果你有宿敌 → 宿敌突破时，野心 +20，直接推高战斗/修炼评分
如果你的道侣死了 → 社交需求大幅上升，触发「寻觅新道侣」或「复仇」
如果你的弟子背叛了 → 忠诚 -30，对所有「收弟子」行动评分 -50%
如果你欠某人的灵石 → 财富需求加速上升
```

### 7.2 因果驱动的行动链

```
一个完整的因果驱动事件链：

1. NPC 甲在后台修炼（效用评分：修炼 60 分 > 其他）
2. 甲突破到金丹 1 层 → 因果网络广播「甲突破金丹」
3. NPC 乙（甲的宿敌）收到广播 →
   乙的修炼评分 +30（野心驱动），突破需求 +40
   → 乙开始疯狂修炼
4. NPC 丙（甲的道侣）收到广播 →
   丙的突破需求 +15（不想被道侣甩下）
   → 丙也开始修炼，但可能因为心魔（急功近利）失败
5. 丙修炼失败，被心魔反噬 → 甲收到「道侣重伤」
   → 甲的社交需求 +60，行动改为「寻找疗伤丹药」
   → 甲无法继续修炼，陷入丹药交易/采集的循环中
   -------------------------------------------
   玩家视角：你路过一个坊市，看到一个金丹修士正在焦急地四处寻找疗伤丹，
   他的道侣在旁边气息微弱。你问了一句，他说：「她是为了不拖我后腿……」
   你手里正好有一颗。你要给他吗？
```

## 八、玩家不可见的异步推演

在玩家离线或不在当前房间时，所有 B 态 NPC 的效用 AI 在后台批量推演。推演结果不逐帧记录——只记录「关键事件」，供玩家回府时通过洞府日志、坊市传闻、NPC 对话等渠道感知。

### 8.1 异步推演流程

```
推演周期：每个游戏时辰（现实 20 秒）对每个 B 态 NPC 推演 1 步

1. 读取 NPC 的：
   · 当前需求条值（随时间衰减）
   · 当前所处房间
   · 因果网络子图
   · 性格参数

2. 枚举可选行动 → 评分 → 选最高分

3. 执行行动：
   · 修炼 → 积累修为（按公式）
   · 移动 → 切换到目标房间
   · 战斗 → 按战斗属性 × 概率决定胜负
   · 交易 → 物品从一方移到另一方
   ……

4. 触发连锁检查：
   · NPC 修为满了 → 尝试突破 → 成功/失败
   · NPC 死了 → 因果线广播 → 影响相关 NPC
   · NPC 移动到了玩家房间 → 从 B 态切换到 A 态（玩家可见）

5. 记录关键事件日志（非结构化的简短描述）
```

### 8.2 推演优化

```
批量推演（Batch Tick）：
  不是每个 NPC 独立推演 → 而是所有 NPC 在一个批量循环中处理
  这样比如 NPC 甲攻击 NPC 乙，乙在同一批次中就能响应（而不是等到下一轮）

层级优先级：
  高境 NPC（地仙+）每 10 个时辰推演 1 步（他们做事慢）
  中境 NPC（金丹-大乘）每 1 个时辰推演 1 步
  低境 NPC（炼气-筑基）每 1 个时辰推演 1 步
  所有 B 态 NPC 共享一个推演管理器，避免创建大量异步任务
  
事件中心（Event Hub）：
  所有推演产出的「关键事件」进入一个共享事件池
  玩家的世界状态查询（坊市变化、NPC 行踪、洞府日志）从此池中提取
  事件池有上限（最近 1000 条），旧事件自动清除
```

## 九、玩家可见的行为表现

效用 AI 的推演结果不是给开发者看的——要转化为玩家能**感知到**的世界变化。

### 9.1 直接可见

| 场景 | NPC 行为 | 玩家感知 |
|------|----------|----------|
| 走进同一个房间 | NPC 正在修炼/采集/交易 | 房间列表显示 NPC 名字 + 状态 |
| 在坊市 | NPC 叫卖、购买 | 「你听到一个散修在和药师讨价还价」 |
| 在野外 | NPC 战斗 | 「前方有灵力波动——两个修士正在交手」 |
| 回洞府 | 日志 | 「天逆历 47 年，一名金丹修士曾在你的洞府外逗留三日」 |

### 9.2 间接感知（传闻系统）

坊市、宗门、茶馆中有「传闻」——不是任务，是纯叙事文本：

```
传闻生成规则：
  每当系统推演出一个「关键事件」，生成一条对应的传闻
  玩家进入有 NPC 的房间时，随机展示 1-2 条

传闻示例：
  「听说天元城的王散修上个月筑基成功了——他拜了个金丹师父。」
  （背后：NPC 王散修在后台推演中触发拜师事件 → 被高境 NPC 接受 → 突破）

  「青木宗的护山大阵被人偷了阵图，宗主大怒，正悬赏捉拿内鬼。」
  （背后：一个 NPC 在后台触发了『潜入宗门』行动 → 成功窃取阵图 → 卖了）

  「最近北边深山里有个元婴老怪坐化了，洞府的位置被一个采药人发现。」
  （背后：NPC 元婴老怪在后台推演中寿元耗尽 → 陨落 → 洞府成为空房间 → 
         另一个 NPC 在采集时偶然经过 → 触发奇遇 → 系统生成了这条传闻）
```

## 十、AI 的修炼与成长

### 10.1 修炼自主权

NPC 的修炼与玩家完全同公式（SYSTEMS.md 第三章）。区别仅在于：玩家手动点「修炼」，NPC 由效用评分自动决定「何时修、在哪修、修多久」。

```
NPC 修炼决策：

  修炼评分 > 其他行动评分 → 本回合修炼
  修炼位置选择：
    · 评分中包含环境因子：灵气浓度高的房间 +30% 修炼评分
    · NPC 会主动移动到灵气更浓的房间
    · 如果一个 NPC 多次经过某个高灵脉房间，它可能在此「定居」

  修炼持续时间：
    · 一次修炼至少持续 1 时辰
    · 如果修炼评分持续最高 → 连续修炼多时辰
    · 需求衰减机制保证它不会「修永无止境」：
       修炼需求下降 → 修炼评分下降 → 其他行动分超过修炼 → 停止修炼
```

### 10.2 属性成长

NPC 的六维主属性（体魄/神识/气海/道心/悟性/根骨）同玩家公式增长：

```
每突破 1 小层：体魄/神识/气海 +1~3（随机，受天赋影响）
每突破 1 大境界：+10~30
道心：通过事件改变（被背叛 -10，收弟子 +5，突破失败 -3……）
悟性：炼丹/炼器/学功法时小概率 +1
根骨：基本固定，仅奇遇可改变
```

### 10.3 功法学习

```
NPC 功法获取途径（按效用评分驱动）：
  · 宗门藏经阁：宗门弟子定期去兑换功法 → 学习 → 修炼
  · 坊市购买：财富需求低 + 修为需求高时 → 买功法
  · 拜师获得：师父传授 → 忠诚度影响传授概率
  · 遗迹发现：探险行动中偶然获得
  · 自创功法：太乙期以上，AI 尝试融合已有功法

学习限制：
  · NPC 只能同时修炼 1 本功法
  · 切换功法有冷却（和玩家相同规则）
  · 灵根匹配影响功法选择（火灵根 NPC 不会主动学水系功法）
```

### 10.4 探索行为

```
NPC 探索由「求知欲」驱动，表现在行动评分中：

  未知房间偏好：如果一个房间 NPC 从没去过 → 移动评分 +15%
  危险评估：高危险房间（有高境界妖兽）→ 风险评估因子拉低总分
  宝藏传闻：如果 NPC 听到「某地有遗迹」→ 移动评分 +50
  采集需求：财富需求高 + 知道有资源的房间 → 前往采集
```

## 十一、AI 的社交与情感

### 11.1 情感状态机

每个 NPC 有一个**情感状态**，不是需求条的一部分——是持续性的心理状态。情感状态影响所有行动评分的权重。

| 情感 | 触发 | 持续 | 影响 |
|------|------|------|------|
| **满足** | 突破成功 / 道侣相伴 / 财富充裕 | 3-10 天 | 修炼 +10%，社交偏好 +20% |
| **焦虑** | 修为卡住 / 灵石见底 / 宿敌突破 | 5-15 天 | 修炼 +20%（更努力），但道心临时 -5 |
| **悲恸** | 道侣/师父/弟子死亡 | 30-100 天 | 修炼 -30%，社交需求大涨（寻求安慰），易触发「复仇」执念 |
| **愤怒** | 被背叛 / 被击败 / 弟子被杀 | 10-30 天 | 战斗评分 +50%，理智 -10，高风险行动权重翻倍 |
| **骄傲** | 收弟子 / 宗门壮大 / 获得奇遇 | 10-20 天 | 修炼 -10%（自满），但接受低境界 NPC 拜师概率 +30% |
| **恐惧** | 濒死经历 / 目睹强者战斗 / 被追杀 | 5-30 天 | 修炼 +10%（为了自保），但冒险行动 -50% |
| **好奇** | 听到传闻 / 发现未知房间 / 虚空游商出现 | 1-5 天 | 探索 +40%，交易 +20% |
| **平静** | 默认状态 / 情感消退后 | — | 无特殊影响 |

### 11.2 情感演化

```
情感不是开关——是渐变：

  焦虑（持续 10 天）
    ↓
  Day 1-3: 强焦虑 → 修炼狂飙
  Day 4-6: 轻度焦虑 → 修炼略高，但开始考虑「要不要换个方法」
  Day 7-10: 焦虑消退 → 如果在消退前没有突破 → 可能转为「沮丧」
  
  沮丧（焦虑消退 + 无突破）→ 修炼 -15%，道心临时 -10
    ↓ 如果此时遇到奇遇/高人指点 → 沮丧立即消散 → 转为「希望」
    ↓ 希望持续 3 天，期间修炼 +25%
```

### 11.3 NPC 间社交

NPC 之间的社交不是「系统广播」——是效用评分驱动下主动发起的交互：

```
社交行动池：

  主动拜访：社交需求 > 60 且知道目标位置 → 前往目标房间
    · 如果对方社交需求也高 → 两人「相谈甚欢」，双方社交需求 -30
    · 如果对方有仇恨 → 可能触发冲突

  送礼：社交需求高 + 财富充裕 → 给关系好的 NPC 送礼
    · 收礼方好感度 +10~30
    · 可能触发「回礼」行动链

  切磋：胆量 > 60 + 修为需求高 + 对方境界相近 → 发起切磋
    · 胜负由战斗公式决定
    · 败方可能产生「超越执念」

  结交道侣：好感度 > 80 + 双方无道侣 → 概率触发
    · 成功后建立「道侣」因果线
    · 双方修炼各 +5%（互相扶持）

  背叛：低忠诚 + 高野心 + 对方有好东西 → 
    · 可能偷窃、出卖情报、甚至在秘境中下黑手
    · 被发现则因果线从「道友」转为「仇敌」
```

### 11.4 喜怒哀乐的文字表现

玩家通过文本感知 NPC 的情感，不需要 UI 标签：

```
满足：「她嘴角带着一抹淡淡的笑意，像是刚突破了某个瓶颈。」
焦虑：「他不停地踱步，偶尔看一眼怀里的丹方残页。」
悲恸：「她的眼神是空的——那种空，和死了的人一样。」
愤怒：「他握剑的手在抖。不是因为恐惧。」
骄傲：「他看你的目光里，有一种过来人的温和与距离。」
恐惧：「他说话的声音压得很低，眼睛却不停地往门外瞥。」
```

## 十二、AI 在世界中的生成

### 12.1 初始世界种子

游戏首次启动时（创建角色后），系统生成「初始世界」。不是空白的——**世界中已经有人了**。

```
初始 NPC 生成数量：

  按地图房间数 × 密度系数：
    坊市房间：2-5 个 NPC（多，因为人流大）
    野外房间：0-2 个 NPC（取决于危险度和资源丰富度）
    宗门房间：5-20 个 NPC（宗门弟子+长老+掌门）
    秘境入口：0-1 个 NPC（偶尔有人徘徊）

  以 50 个地图房间为例 → 初始约 80-150 个 NPC

  初始 NPC 的境界分布：
    炼气：40%（大多数人是底层）
    筑基：25%
    金丹：15%
    元婴：10%
    化神：6%
    合体及以上：4%（是世界的「顶层架构」）
```

### 12.2 NPC 创建流程

```
1. 分配身份：
   · 70% 散修（无宗门，自由移动）
   · 20% 宗门弟子（有宗门归属，行动受宗门影响）
   · 8% 宗门长老/掌门
   · 2% 特殊身份（虚空游商、隐世高人、黑市中间人）

2. 随机属性：
   · 体魄/神识/气海/悟性/根骨：5-18 间随机
   · 道心：30-70 间随机
   · 灵根：金(19%)/木(19%)/水(19%)/火(19%)/土(19%)/变异(5%)
   · 纯度：下品(55%)/中品(25%)/上品(12%)/极品(6%)/天品(2%)

3. 随机性格（5 维）：
   · 胆量/野心/勤勉/忠诚/贪婪：各 0-100

4. 随机功法（至少 1 本）：
   · 按灵根匹配功法五行
   · 境界越高，功法数量越多（金丹可同时学 2-3 本）

5. 随机初始物品：
   · 凡品材料 3-10 件
   · 境界匹配的装备 1-3 件
   · 灵石：境界 × 100~500

6. 初始因果网络：
   · 宗门 NPC → 与同门建立「同门」关系
   · 每个 NPC → 0-2 个随机因果（道侣/宿敌/师徒/道友）
   · 概率不高——大多数人的世界开始时是简单的
```

### 12.3 持续的 NPC 生成

```
世界不是开服后就「固定」的：

  新的 NPC 来源：
    1. 修士子嗣：道侣 NPC 在推演中有极小概率（万年历法）生育 → 
       新生儿 15 游戏年后成年 → 成为新 NPC。继承父母灵根（混合概率）。
    2. NPC 收徒：高境 NPC 主动出去寻找有天赋的低境 NPC → 
       如果没有合适的 → 触发「收徒」行动时可能从零生成一个新 NPC。
    3. 宗门招新：宗门定期在坊市「招新」→ 
       如果响应者不够 → 系统在坊市生成 1-3 个炼气期散修。
    4. 天道异变：天魔入侵/灵气潮汐等事件中可能生成一批临时 NPC →
       事件结束后部分留下成为永久 NPC。
    5. 虚空游商：每次游商造访后可能「留下」一个 NPC（比如游商收了徒弟）。

  NPC 的自然消亡：
    · 寿元耗尽（最常见）
    · 战斗死亡（被妖兽或 NPC 杀死）
    · 渡劫失败（灰飞烟灭）
    · 禁术反噬（摄魂/诅咒/血祭的代价）
    · 被玩家杀死（玩家也是世界的一部分）

  动态平衡：
    初始 150 个 NPC → 随时间自然消亡 → 新 NPC 生成 →
    稳定在 120-200 个之间（取决于世界事件密度）
```
